12 - Ecuația de gradul I cu o singură necunoscută

Definiție

Este o ecuație care, după transformare, are forma a⋅x=ba \cdot x = ba⋅x=b, în care aaa și bbb sunt nr. cunoscute și xxx este necunoscuta.

Transformări posibile​

  • dezvoltare sau factorizare a celor doi membri
  • adunarea aceleași sume algebrice în ambii membri
  • înmulțirea sau împărțirea celor doi membrii cu același număr (≠0\ne 0=0)
Important

Dacă a≠0a \ne 0a=0, atunci ecuația a⋅x=ba \cdot x = b a⋅x=b are ca soluție x=bax = \Large \frac{b}{a}x=ab​

Exemple​

cerințǎ

Fie ecuația 4x+24=−6x+34x+24=-6x+34x+24=−6x+3

adaugăm (−24)(-24)(−24) în ambii membri:

4x+24+(−24)=−6x+3+(−24)4x + 24 + (-24) = -6x + 3 + (-24)4x+24+(−24)=−6x+3+(−24)

obținem 4x=−6x+3−244x = -6x + 3 - 244x=−6x+3−24

adaugăm +6x+6x+6x în ambii membri:

4x+6x=−6x+3+(−24)+6x4x + 6x = -6x + 3 + (-24) + 6x4x+6x=−6x+3+(−24)+6x

⇒10x=−21\Rightarrow 10x = -21⇒10x=−21, unde a = 10 și b = -21

x = −2110\Large \frac{-21}{10}10−21​

x=−2,1\boldsymbol{x = -2,1}x=−2,1

cerințǎ

Fie ecuația (2x+7)⋅(3x−5)=0(2x+7)\cdot (3x-5) = 0(2x+7)⋅(3x−5)=0

Primul membru este un produs de 2 factori de gradul I; al doilea membru este nul.

Fiecare valoare care anulează unul dintre cei 2 factori din membrul întâi este o soluție a ecuației produs.

Primul factor:

2x+7=0⇒2x=−7⇒x=−72 2x + 7 = 0 \Rightarrow 2x = -7 \Rightarrow \boldsymbol{x = -\Large \frac{7}{2}}2x+7=0⇒2x=−7⇒x=−27​

Al doilea factor:

3x−5=0⇒3x=5⇒x=53 3x - 5 = 0 \Rightarrow 3x = 5 \Rightarrow \boldsymbol{x = \Large \frac{5}{3}}3x−5=0⇒3x=5⇒x=35​

Etapele de rezolvare​

1. Alegem necunoscuta: folosim o literă pentru desemnarea necunoscutei.

Notăm cu x măsura unghiului B^\hat{B}B^

2. Precizăm unitățile și condițiile impuse de necunoscută.

Unitate de măsură: ∘\circ∘ (grade)

0 < x < 180

  1. Punerea în ecuație a problemei.
C^=2⋅B^⇒C^=2⋅x\hat{C} = 2 \cdot \hat{B} \Rightarrow \hat{C} = 2 \cdot xC^=2⋅B^⇒C^=2⋅x
  • Spunem că suma unghiurilor este egală cu 180∘180\circ180∘

A^+B^+C^=180∘,unde{A^=117∘B^=xC^=2xA^+B^+C^=117∘+x+2x\hat{A} + \hat{B} + \hat{C} = 180\circ, unde \\[8pt] \begin{cases} \hat{A} = 117\circ\\ \hat{B} = x\\ \hat{C} = 2x\\ \end{cases} \boldsymbol{\hat{A} + \hat{B} + \hat{C} = 117\circ + x + 2x}A^+B^+C^=180∘,unde⎩⎨⎧​A^=117∘B^=xC^=2x​A^+B^+C^=117∘+x+2x

  1. rezolvarea

117∘+x+2x=180∘⇒117∘+3x=180∘3x=180∘−117∘3x=63∘x=63:3=21∘117\circ + x + 2x = 180\circ \Rightarrow\\[8pt] 117\circ + 3x = 180\circ\\[8pt] 3x = 180\circ - 117\circ\\[8pt] 3x = 63\circ\\[8pt] \boldsymbol{x = 63 : 3 = 21\circ}117∘+x+2x=180∘⇒117∘+3x=180∘3x=180∘−117∘3x=63∘x=63:3=21∘
