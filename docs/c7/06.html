C5 - Mulțimea nr. reale

Incluziune​

De Retinut
N⊂Z⊂Q⊂R\Large \mathbb{N}\subset\mathbb{Z}\subset\mathbb{Q}\subset\mathbb{R}N⊂Z⊂Q⊂R

  • Notăm N\mathbb{N}N mulțimea numerelor întregi naturale: N={0;1;2;3;…}\mathbb{N} = \{0 ; 1 ; 2 ; 3 ; \ldots\}N={0;1;2;3;…}.

  • Notăm Z\mathbb{Z}Z mulțimea numerelor întregi: Z={…;−3;−2;−1;0;1;2;3;…}\mathbb{Z} = \{\ldots ; -3 ; -2 ; -1 ; 0 ; 1 ; 2 ; 3 ; \ldots\}Z={…;−3;−2;−1;0;1;2;3;…}.

  • Notăm Q\mathbb{Q}Q mulțimea numerelor raționale: adică numerele care se scriu sub forma pq\Large \dfrac{p}{q}qp​, cu p∈Zp\in\mathbb{Z}p∈Z și q∈N∗q\in\mathbb{N}^*q∈N∗ (mulțimea numerelor naturale nenule).

Observație: toate numerele zecimale sunt raționale, deci mulțimea numerelor zecimale este inclusă în Q\mathbb{Q}Q.

  • Notăm R∖Q\mathbb{R} \setminus \mathbb{Q}R∖Q mulțimea numerelor iraționale, acele numere care, scrise zecimal, au o infinitate de cifre în dreapta virgulei care nu se repetǎ periodic.

  • Notăm R\mathbb{R}R mulțimea numerelor reale, adică mulțimea tuturor numerelor.

Cea mai mică mulțime​

Definitie
  • Dacă un număr xxx aparține unei mulțimi AAA, scriem: x∈Ax\in Ax∈A.
  • Dacă un număr xxx nu aparține unei mulțimi AAA, scriem: x∉Ax\notin Ax∈/A.

Vom desemna prin „cea mai mică mulțime” a unui număr acea mulțime din ierarhia mulțimilor numerelor din care face parte, începând cel puțin de la mulțimea Q\mathbb{Q}Q — deoarece orice număr zecimal este rațional.

Exemplu: Cea mai mică mulțime a numărului 515\Large \dfrac{51}{5}551​ este Q\mathbb{Q}Q (nu este întreg). Scriem: 515∈Q\Large \dfrac{51}{5} \in \mathbb{Q}551​∈Q.

Astfel, cea mai mică mulțime a lui 9\sqrt{9}9​ este N\mathbb{N}N, scriem: 9∈N\sqrt{9}\in\mathbb{N}9​∈N.

Proprietatea lui radical din 2​

2\sqrt{2}2​ nu este rațional.

Demonstrație. Presupunem, prin reducere la absurd, că 2=pq\sqrt{2} = \Large \dfrac{p}{q}2​=qp​ cu p,q∈Zp,q\in\mathbb{Z}p,q∈Z, q≠0q\ne 0q=0, iar fracția este ireductibilă (p și q nu au divizori comuni >1). Atunci:

2q2=p2.2q^2 = p^2.2q2=p2.

Din egalitate rezultă că p2p^2p2 este par, deci ppp este par: există k∈Zk\in\mathbb{Z}k∈Z cu p=2kp=2kp=2k. Înlocuind:

2q2=4k2⇒q2=2k2.2q^2 = 4k^2 \quad\Rightarrow\quad q^2 = 2k^2.2q2=4k2⇒q2=2k2.

Deci q2q^2q2 este par, ceea ce implică că qqq este par. Astfel ppp și qqq sunt amândoi pare, deci au factorul 2 în comun, contradicând ipoteza de ireductibilitate. Contradicția arată că presupunerea inițială este falsă; deci 2∉Q\sqrt{2}\notin\mathbb{Q}2​∈/Q.

Mai multe scrieri pentru același număr​

Un număr real poate fi reprezentat în mai multe feluri. De exemplu:

515=5,10,5=10,2=1,02⋅101\Large \dfrac{51}{5} = \Large \dfrac{5,1}{0,5} = 10,2 = 1,02\cdot10^1551​=0,55,1​=10,2=1,02⋅101

sau:

43=1,33…\Large \dfrac{4}{3} = 1,33\ldots34​=1,33…

Indiferent de scriere, este același număr.

Important

Observații:

  • 515\Large \dfrac{51}{5}551​ nu este doar rațional; este și real.

  • 9\sqrt{9}9​ nu este doar real; este și un întreg natural, deoarece 9=3\sqrt{9}=39​=3.

Aceste proprietăți sunt ale numărului în sine, nu ale scrierii sale.

De reținut

Dacă o mulțime este urmată de o steluță la exponent, ea conține toate elementele mulțimii în afară de 0. De exemplu:

N∗={1,2,3,…}.\mathbb{N}^* = \{1,2,3,\ldots\}.N∗={1,2,3,…}.

Incluziunea a două mulțimi oarecare​

Definitie

Fie AAA și BBB două mulțimi. Spunem că AAA este inclusă în BBB și notăm A⊂BA\subset BA⊂B dacă toate elementele din AAA sunt în BBB.

Dacă vrem să eliminăm unul sau mai multe numere dintr-o mulțime, scriem operația de diferență: ∖{…}\setminus\{\ldots\}∖{…}.

Exemple:

  • Q∖{13}\Large\mathbb{Q}\setminus\{\dfrac{1}{3}\}Q∖{31​} desemnează mulțimea numerelor raționale fără 13\Large \dfrac{1}{3}31​.

  • R∖Q\Large\mathbb{R}\setminus\mathbb{Q}R∖Q este mulțimea numerelor reale din care au fost eliminate toate numerele raționale; această mulțime este numită mulțimea iraționalelor.

Mulțimea vidă​

De Retinut

Există o mulțime fără elemente, numită mulțimea vidă, notată:

∅\Large\varnothing∅

Paritate​

Definiție

Multiplii lui 2 sunt numerele întregi pare. Celelalte numere întregi sunt impare.

Prin definiție:

  • numerele întregi pare au forma 2n2n2n, cu n∈Zn\in\mathbb{Z}n∈Z;
  • numerele întregi impare au forma 2n+12n+12n+1.
Proprietatea 1
  • Pătratul unui număr întreg par este par.
  • Pătratul unui număr întreg impar este impar.
Proprietatea 2

Fie xxx un întreg.

  • Dacă x2x^2x2 este par, atunci xxx este par.
  • Dacă x2x^2x2 este impar, atunci xxx este impar.
