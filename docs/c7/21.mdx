# 21 - Paralelogramul
---

## Definiție și proprietăți

:::tip
Un paralelogram este un patrulater ale cărui laturi opuse sunt paralele.
:::

:::info Proprietăți
Un paralelogram are:

* laturi opuse`paralele` și de aceeași lungime
* diagonalele se `intersectează` în `mijlocul` lor
* unghiurile opuse sunt `egale`
* Punctul de intersecție al diagonalelor este `centrul de simetrie` al paralelogramului
:::

> Spunem că ABCD este un paralelogram cu centrul O

* $(AB)//(DC); (AD)//(BC)$
* $AB = DC; AD = BC$
* $O \in [AC]$ și $OA = OC$
* $O \in [BD]$ și $OB = OD$
* $\measuredangle ABC = \measuredangle ADC$; $\measuredangle BCD = \measuredangle BAD$

## Recunoașterea unui paralelogram
:::danger De reținut
* Dacă un patrulater are **laturi opuse paralele**, atunci este un paralelogram.
* Dacă un patrulater are **diagonale** care **se intersectează la mijloc**, atunci este un paralelogram.
* Dacă un patrulater are **două laturi paralele** și de **aceeași lungime**, atunci este un paralelogram.
:::

## Translație

:::warning Definiție DEX
> **translație, translații - substantiv feminin** 

În matematică, translația este transformare geometrică punctuală, determinată de un vector, astfel încât `unui punct` din spațiu **să-i corespundă** un `alt punct`.
:::

Avem un triunghi $\triangle ABC$ și un punct $O$.
Aplicăm o mișcare de translație triunghiului din punctul $A$ spre $O$.
> **exemplu practic:** teoria se utilizează în proiectarea software-ului ecranelor digitale

:::info Atenție
Triunghiul $OMP$ este imaginea triunghiului $ABC$ prin translația din $A$ spre $O$
:::

* Patrulaterul $AOMB$ este un paralelogram. \
  Avem: $(BM)//(AO);BM = AO$ 
  > pornind din punctul $B$ spre $M$ mergem în același sens ca de la $A$ la $O$

* Patrulaterul $AOPC$ este un paralelogram.
  Avem: $(CP)//(AO); CP = AO$ 
  > pornind din punctul $C$ spre $P$ mergem în același sens ca de la $A$ la $O$
<br />

:::info Atenție
Triunghiul OMP este **identic** triunghiului $ABC$
:::

Cunoașterea a două puncte este suficientă pentru a defini o translație.
Avem punctele $A$ spre $O$, perechea $(A, O)$ este *grupul de direcție* al translației.

$
\begin{cases}
\text {un punct inițial A} \\[5pt]
\text {un punct final (extremitate) O}
\end{cases}, \text {unde sensul de deplasare este important}
$

### Imaginea unui punct

**Caz general:** $P ∉ (AO)$

Pentru a construi imaginea $P'$ a punctului $P$:

* trasăm dreapta $xy$, paralelă cu $(AO)$ și care trece prin $P$
* plasăm pe această dreaptă punctul $P'$ astfel încât $PP' = AO$ și sensul de la $P$ la $P'$ să fie același ca de la $A$ la $O$

$[A'B'] // [AB]; [B'C'] // [BC]$ 

$A'B' = AB; B'C' = BC$ 

$\hat {A'} = \hat{A}; \hat{B'} = \hat{B}; \hat{C'} = \hat{C}$ 

Aria $\triangle A'B'C'$ = Aria $\triangle ABC$
