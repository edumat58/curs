G5 - Linia mijlocie în triunghi

Definiție​

Definiție

Segmentul care unește mijloacele a două laturi ale unui triunghi se numește linie mijlocie a triunghiului.

Dacă M,N,PM, N, PM,N,P sunt mijloacele laturilor triunghiului ABCABCABC,
atunci segmentele MN,NP,MPMN, NP, MPMN,NP,MP sunt liniile mijlocii ale triunghiului.

Proprietate​

Proprietate

Linia mijlocie determinată de mijloacele a două laturi ale unui triunghi este paralelă cu a treia latură și are lungimea egală cu jumătate din lungimea acesteia.

Astfel, pentru triunghiul ABCABCABC:

MN∥BC,NP∥AB,MP∥ACMN \parallel BC,\quad NP \parallel AB,\quad MP \parallel ACMN∥BC,NP∥AB,MP∥AC

și

MN=BC2,NP=AB2,MP=AC2MN = \frac{BC}{2}, \quad NP = \frac{AB}{2}, \quad MP = \frac{AC}{2}MN=2BC​,NP=2AB​,MP=2AC​

sau echivalent:

BC=2⋅MN,AB=2⋅NP,AC=2⋅MPBC = 2\cdot MN, \quad AB = 2\cdot NP, \quad AC = 2\cdot MPBC=2⋅MN,AB=2⋅NP,AC=2⋅MP

Aplicații​

Exemplul 1​

Problemă

Fie triunghiul ABCABCABC în care MNMNMN, NPNPNP, MPMPMP sunt liniile mijlocii.
Știind că BC=8 cmBC = 8\text{ cm}BC=8 cm, AC=7 cmAC = 7\text{ cm}AC=7 cm și NP=3 cmNP = 3\text{ cm}NP=3 cm,
să se calculeze perimetrul triunghiului MNPMNPMNP.

Rezolvare:​

MN=BC2=82=4 cmMN = \frac{BC}{2} = \frac{8}{2} = 4\text{ cm}MN=2BC​=28​=4 cm MP=AC2=72=3,5 cmMP = \frac{AC}{2} = \frac{7}{2} = 3{,}5\text{ cm}MP=2AC​=27​=3,5 cm NP=3 cmNP = 3\text{ cm}NP=3 cm

Perimetrul triunghiului MNPMNPMNP este:

PMNP=MN+NP+MP=4+3+3,5=10,5 cmP_{MNP} = MN + NP + MP = 4 + 3 + 3{,}5 = 10{,}5\text{ cm}PMNP​=MN+NP+MP=4+3+3,5=10,5 cm
De reținut
PMNP=PABC2P_{MNP} = \frac{P_{ABC}}{2}PMNP​=2PABC​​

Exemplul 2​

Problemă

Fie paralelogramul ABCDABCDABCD, în care S,T,U,VS, T, U, VS,T,U,V sunt mijloacele laturilor.
Se cere să se arate că patrulaterul STUVSTUVSTUV este paralelogram.

Rezolvare:​

În △ABD\triangle ABD△ABD, punctele VVV și SSS sunt mijloacele laturilor ADADAD și ABABAB,
deci SVSVSV este linie mijlocie:

VS∥BD,VS=BD2VS \parallel BD, \quad VS = \frac{BD}{2}VS∥BD,VS=2BD​

În △CBD\triangle CBD△CBD, punctele UUU și TTT sunt mijloacele laturilor CDCDCD și CBCBCB,
deci UTUTUT este linie mijlocie:

UT∥BD,UT=BD2UT \parallel BD, \quad UT = \frac{BD}{2}UT∥BD,UT=2BD​

Rezultă:

  • VS∥BDVS \parallel BDVS∥BD și UT∥BD⇒VS∥UTUT \parallel BD \Rightarrow VS \parallel UTUT∥BD⇒VS∥UT
  • VS=BD2VS = \frac{BD}{2}VS=2BD​ și UT=BD2⇒VS=UTUT = \frac{BD}{2} \Rightarrow VS = UTUT=2BD​⇒VS=UT

Prin urmare:

STUV este paralelogram.STUV \text{ este paralelogram.}STUV este paralelogram.
