C9 - Ecuația x² = a

Cazuri​

Pentru a>0a > 0a>0, ecuația x2=ax^2 = ax2=a are două soluții distincte:

x=asaux=−ax = \sqrt{a} \quad \text{sau} \quad x = -\sqrt{a}x=a​saux=−a​

Pentru a=0a = 0a=0, ecuația x2=ax^2 = ax2=a admite soluția:

x=0x = 0x=0

Pentru a<0a < 0a<0, ecuația x2=ax^2 = ax2=a nu are soluții reale.

Transformări echivalente​

Important

Multe dintre ecuații sunt prezentate într-o altă formă, dar prin transformări echivalente, putem obține ecuații de forma x2=ax^2 = ax2=a.

Aceste transformări se obțin, ca și în cazul altor ecuații studiate anterior, dacă:

  • adunăm sau scădem la ambii membri ai ecuației același număr real;
  • înmulțim sau împărțim ambii membri ai ecuației cu același număr real nenul.

x2=32⇒x=32 sau x=−32x^2 = 32 \Rightarrow x = \sqrt{32} \text{ sau } x = -\sqrt{32}x2=32⇒x=32​ sau x=−32​x=42 sau x=−42x = 4\sqrt{2} \text{ sau } x = -4\sqrt{2}x=42​ sau x=−42​S={−42,42}S = \{-4\sqrt{2}, 4\sqrt{2}\}S={−42​,42​}
