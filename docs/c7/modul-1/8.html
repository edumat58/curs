C13 - Puterea cu exponent număr natural a unui număr întreg nenul

1. Definiție

Puterea a n-a a unui număr întreg nenul a este numărul:

an=a⋅a⋅a⋯a⏟_n factori,a^n = \underbrace{a \cdot a \cdot a \cdots a}\_{n\ \text{factori}},an=a⋅a⋅a⋯a​_n factori,

unde

  • a = baza,
  • n = exponent, n ∈ ℕ.
cerințǎ

Se notează cu ana^nan produsul dintre nnn factori egali cu aaa.

Exemple​

  • (+6)2=(+6)⋅(+6)=+36(+6)^2 = (+6)\cdot(+6) = +36(+6)2=(+6)⋅(+6)=+36
  • (−2)3=(−2)⋅(−2)⋅(−2)=−8(-2)^3 = (-2)\cdot(-2)\cdot(-2) = -8(−2)3=(−2)⋅(−2)⋅(−2)=−8
  • (−3)4=(−3)⋅(−3)⋅(−3)⋅(−3)=+81(-3)^4 = (-3)\cdot(-3)\cdot(-3)\cdot(-3) = +81(−3)4=(−3)⋅(−3)⋅(−3)⋅(−3)=+81

2. Observații importante

  • a0=1a^0 = 1a0=1, pentru orice a∈Z, a≠0a \in \mathbb{Z},\ a\neq 0a∈Z, a=0
    Exemplu: (−5)0=1(-5)^0 = 1(−5)0=1

  • 1n=11^n = 11n=1, pentru orice n∈Nn \in \mathbb{N}n∈N

  • a1=aa^1 = aa1=a
    Exemplu: (−80)1=−80(-80)^1 = -80(−80)1=−80

  • 0n=00^n = 00n=0, pentru orice n∈Nn \in \mathbb{N}n∈N
    Exemplu: 072=00^{72} = 0072=0

  • 000^000 nu are sens

3. Semnul unei puteri

3.1. Dacă baza este pozitivă​

definitie

Orice putere a unui număr pozitiv este pozitivă.

Exemplu:
(+3)2=+9(+3)^2 = +9(+3)2=+9

3.2. Dacă baza este negativă​

  • Dacă exponentul este par, rezultatul este pozitiv
    Exemplu: (−4)2=+16(-4)^2 = +16(−4)2=+16

  • Dacă exponentul este impar, rezultatul este negativ
    Exemplu: (−2)5=−32(-2)^5 = -32(−2)5=−32

3.3. Atenție la notații!​

cerințǎ

(−a)2≠−a2(-a)^2 \neq -a^2(−a)2=−a2

  • (−a)2=(+a2)(-a)^2 = (+a^2)(−a)2=(+a2)
  • −a2=−(a2)-a^2 = -(a^2)−a2=−(a2)

Exemplu:
(−5)2=+25(-5)^2 = +25(−5)2=+25
−52=−25-5^2 = -25−52=−25

Cazuri speciale pentru (−1)n(-1)^n(−1)n​

  • (−1)n=1(-1)^n = 1(−1)n=1, dacă nnn este par
  • (−1)n=−1(-1)^n = -1(−1)n=−1, dacă nnn este impar

4. Reguli de calcul cu puteri

4.1. Înmulțirea puterilor cu aceeași bază​

an⋅am=an+ma^n \cdot a^m = a^{n+m}an⋅am=an+m

Exemplu:
(−3)2⋅(−3)4=(−3)6=729(-3)^2 \cdot (-3)^4 = (-3)^6 = 729(−3)2⋅(−3)4=(−3)6=729

4.2. Împărțirea puterilor cu aceeași bază​

an:am=an−m,n≥ma^n : a^m = a^{n-m},\quad n\ge man:am=an−m,n≥m

Exemplu:
(−2)15:(−2)10=(−2)5=−32(-2)^{15} : (-2)^{10} = (-2)^5 = -32(−2)15:(−2)10=(−2)5=−32

4.3. Puterea unei puteri​

(an)m=an⋅m(a^n)^m = a^{n\cdot m}(an)m=an⋅m

Exemplu:
[(−2)3]2=(−2)6=64\left[(-2)^3\right]^2 = (-2)^6 = 64[(−2)3]2=(−2)6=64

4.4. Puterea unui produs​

(a⋅b)n=an⋅bn(a\cdot b)^n = a^n \cdot b^n(a⋅b)n=an⋅bn

Exemplu:
[(−2)⋅(+5)]7=(−2)7⋅(+5)7\left[(-2)\cdot(+5)\right]^7 = (-2)^7 \cdot (+5)^7[(−2)⋅(+5)]7=(−2)7⋅(+5)7

4.5. Puterea unui cât​

(a:b)n=anbn(a:b)^n = \frac{a^n}{b^n}(a:b)n=bnan​

Exemplu:
[(−10):(+2)]8=(−10)8(+2)8\left[(-10):(+2)\right]^8 = \dfrac{(-10)^8}{(+2)^8}[(−10):(+2)]8=(+2)8(−10)8​

5. Exemple rezolvate

Exemplul 1​

Calculează: (−2)3+52−(−1)8(-2)^3 + 5^2 - (-1)^8(−2)3+52−(−1)8

Rezolvare:

  • (−2)3=−8(-2)^3 = -8(−2)3=−8
  • 52=255^2 = 2552=25
  • (−1)8=1(-1)^8 = 1(−1)8=1

−8+25−1=16-8 + 25 - 1 = 16−8+25−1=16

Exemplul 2​

(−3)5⋅34⋅(−3)6(-3)^5 \cdot 3^4 \cdot (-3)^6(−3)5⋅34⋅(−3)6

Aducem la aceeași bază 3:

(−3)5⋅34⋅(−3)6=−35+4+6=−315(-3)^5 \cdot 3^4 \cdot (-3)^6 = -3^{5+4+6} = -3^{15}(−3)5⋅34⋅(−3)6=−35+4+6=−315

Exemplul 3​

[3⋅(−6)]2=(−18)2=324[3\cdot(-6)]^2 = (-18)^2 = 324[3⋅(−6)]2=(−18)2=324

Exemplul 4​

(−16)3:(−4)3=[(−16):(−4)]3=43=64(-16)^3 : (-4)^3 = [(-16):(-4)]^3 = 4^3 = 64(−16)3:(−4)3=[(−16):(−4)]3=43=64
