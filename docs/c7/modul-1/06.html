C5 - Mulțimea nr. reale (1)

Incluziune​

De Retinut
N⊂Z⊂Q⊂R\Large \mathbb{N}\subset\mathbb{Z}\subset\mathbb{Q}\subset\mathbb{R}N⊂Z⊂Q⊂R

  • Notăm N\mathbb{N}N mulțimea numerelor întregi naturale: N={0;1;2;3;… } \mathbb{N} = \{0;1;2;3;\dots\}N={0;1;2;3;…}.

  • Notăm Z\mathbb{Z}Z mulțimea numerelor întregi: Z={… ;−3;−2;−1;0;1;2;3;… }\mathbb{Z} = \{\dots;-3;-2;-1;0;1;2;3;\dots\}Z={…;−3;−2;−1;0;1;2;3;…}.

  • Notăm Q\mathbb{Q}Q mulțimea numerelor raționale: adică numerele care se scriu sub forma pq\dfrac{p}{q}qp​, cu p∈Zp\in\mathbb{Z}p∈Z și q∈Z∗q\in\mathbb{Z}^*q∈Z∗.

  • Notăm R\mathbb{R}R mulțimea numerelor reale, adică mulțimea tuturor numerelor.

Observație: toate numerele zecimale sunt raționale, deci mulțimea numerelor zecimale este inclusă în Q\mathbb{Q}Q.

Notăm R∖Q\mathbb{R}\setminus\mathbb{Q}R∖Q mulțimea numerelor iraționale, acele numere care, scrise zecimal, au o infinitate de cifre în dreapta virgulei care nu se repeta periodic.

Notăm N∗\mathbb{N}^*N∗ mulțimea numerelor naturale nenule.

Cea mai mică mulțime​

DEFINIȚIE

Dacă un număr xxx aparține unei mulțimi AAA, scriem: x∈Ax\in Ax∈A.

Dacă un număr xxx nu aparține unei mulțimi AAA, scriem: x∉Ax\notin Ax∈/A.

Vom desemna prin „cea mai mică mulțime” a unui număr acea mulțime din ierarhia mulțimilor numerelor din care face parte, începând cel puțin de la mulțimea Q\mathbb{Q}Q — deoarece orice număr zecimal este rațional.

Exemplu:
Cea mai mică mulțime a numărului 515\dfrac{51}{5}551​ este Q\mathbb{Q}Q (nu este întreg). Scriem:

515∈Q.\dfrac{51}{5}\in\mathbb{Q}.551​∈Q.

Proprietatea lui radical din 2​

2 nu este rațional.\sqrt{2}\ \text{nu este rațional.}2​ nu este rațional.

Demonstrație.
Presupunem, prin reducere la absurd, că 2=pq\sqrt{2} = \dfrac{p}{q}2​=qp​ cu p,q∈Zp,q\in\mathbb{Z}p,q∈Z, q≠0q\neq 0q=0, iar fracția este ireductibilă (adică ppp și qqq nu au divizori comuni >1>1>1).
Atunci:

2q2=p2.2q^2 = p^2.2q2=p2.

Din egalitate rezultă că p2p^2p2 este par, deci ppp este par: există k∈Zk\in\mathbb{Z}k∈Z cu p=2kp=2kp=2k. Înlocuind:

2q2=(2k)2  ⟹  q2=2k2.2q^2 = (2k)^2 \implies q^2 = 2k^2.2q2=(2k)2⟹q2=2k2.

Deci q2q^2q2 este par, ceea ce implică și că qqq este par. Astfel ppp și qqq sunt amândoi pari, deci au factorul 222 în comun, contrazicând ipoteza de ireductibilitate.
Contradicția arată că presupunerea inițială este falsă; deci

2∉Q.\sqrt{2}\notin\mathbb{Q}.2​∈/Q.
