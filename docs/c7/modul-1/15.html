G3 - Patrulaterul convex

Introducere​

Important

  • Patru puncte, oricare trei necoliniare, determină un patrulater.

  • Punctele AAA, BBB, CCC și DDD se numesc vârfurile patrulaterului.

  • Segmentele ABABAB, BCBCBC, CDCDCD și DADADA se numesc laturile patrulaterului.

  • Dreptele care includ laturile patrulaterului se vor numi drepte suport ale laturilor patrulaterului.

Patrulater convex și patrulater concav​

Definitie

Un patrulater este convex dacă, pentru oricare dintre laturi, două vârfuri ale patrulaterului se află de aceeași parte a dreptei sale suport.
În caz contrar, patrulaterul este concav.

Ordinea vârfurilor în numirea patrulaterului este de mare importanță:
când numim un patrulater, enumerăm vârfurile circular.

Elemente ale patrulaterului​

Proprietăți​

În figura alăturată avem patrulaterul ABCDABCDABCD cu elementele:

  • Punctele AAA, BBB, CCC și DDD sunt vârfurile patrulaterului.

    • Vârfurile AAA și BBB, de exemplu, sunt vârfuri vecine.
    • Vârfurile AAA și CCC, de exemplu, sunt vârfuri opuse.

  • Patrulaterul ABCDABCDABCD are laturile: ABABAB, BCBCBC, CDCDCD și DADADA.

    • Laturile ABABAB și CDCDCD, de exemplu, sunt laturi opuse.
    • Laturile ABABAB și BCBCBC, de exemplu, sunt laturi alăturate (au un punct comun).

  • Două vârfuri opuse ale unui patrulater convex determină o diagonală:
    segmentele ACACAC și BDBDBD sunt diagonalele patrulaterului ABCDABCDABCD.

  • Oricare două laturi alăturate ale unui patrulater convex formează câte un unghi.
    Unghiurile patrulaterului ABCDABCDABCD sunt:
    ∡DAB\measuredangle DAB∡DAB, ∡ABC\measuredangle ABC∡ABC, ∡BCD\measuredangle BCD∡BCD și ∡CDA\measuredangle CDA∡CDA.
    Dacă nu este pericol de confuzie, se pot nota cu ∡A\measuredangle A∡A, ∡B\measuredangle B∡B, ∡C\measuredangle C∡C, ∡D\measuredangle D∡D.

    • ∡A\measuredangle A∡A și ∡B\measuredangle B∡B, de exemplu, sunt unghiuri alăturate.
    • ∡A\measuredangle A∡A și ∡C\measuredangle C∡C, de exemplu, sunt unghiuri opuse.

  • Partea comună a interioarelor a două unghiuri opuse formează interiorul patrulaterului convex.

Teoremă​

De retinut

Suma măsurilor unghiurilor interioare ale unui patrulater convex este 360∘360^\circ360∘.

m∡A+m∡B+m∡C+m∡D=360∘m\measuredangle A + m\measuredangle B + m\measuredangle C + m\measuredangle D = 360^\circm∡A+m∡B+m∡C+m∡D=360∘

cerințǎ

Vom determina măsura unghiului AAA.

Rezolvare:

m∡A+m∡B+m∡C+m∡D=360∘m\measuredangle A + m\measuredangle B + m\measuredangle C + m\measuredangle D = 360^\circm∡A+m∡B+m∡C+m∡D=360∘ m∡A=360∘−(85∘+110∘+44∘)=360∘−239∘=121∘m\measuredangle A = 360^\circ - (85^\circ + 110^\circ + 44^\circ) = 360^\circ - 239^\circ = 121^\circm∡A=360∘−(85∘+110∘+44∘)=360∘−239∘=121∘
