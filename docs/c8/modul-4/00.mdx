# C15 - Funcția de gradul I și graficul acesteia

---

:::tip Definiție
$f:\mathbb{D} \to \mathbb{R}$, $f(x) = ax + b$, unde $a, b \in \mathbb{R}$ și $\mathbb{D}$ este mulțime finită a lui $\mathbb{R}$, se numește funcție de gradul I.

Graficul unei funcții de gradul I este o dreaptă.
:::

:::danger Atenție
Deoarece o dreaptă este determinată de două puncte distincte, pentru a reprezenta geometric
graficul funcției f: ℝ → ℝ, f(x) = ax + b, unde a, b ∈ ℝ, avem nevoie de cel puțin două puncte.

> Ca să fim siguri că am lucrat corect vom lua trei puncte.
