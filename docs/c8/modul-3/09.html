G8.1 - Trunchi de piramidă regulată

Observație​

cerințǎ

În Figura 18 este desenat un trunchi de piramidă regulată cu baza triunghi echilateral.

  • a) Dacă aria laterală a piramidei din care provine trunchiul este egală cu 100 cm2100\text{ cm}^2100 cm2 și aria laterală a piramidei care se elimină este egală cu 20 cm220\text{ cm}^220 cm2, determină aria laterală a trunchiului de piramidă.
  • b) Dacă volumul piramidei din care provine trunchiul este egal cu 100 cm3100\text{ cm}^3100 cm3 și volumul piramidei care se elimină este egal cu 20 cm320\text{ cm}^320 cm3, determină volumul trunchiului de piramidă.

Important​

important
  • Aria laterală a unui trunchi de piramidă regulată se poate determina ca diferență între aria laterală a piramidei mari și aria laterală a piramidei mici.
    • Alat.tr.de pir.=Alat.P−Alat.p\mathcal{A}_{\text{lat.tr.de pir.}} = \color{red}{\mathcal{A}_{\text{lat.P}}} - \color{blue}{\mathcal{A}_{\text{lat.p}}}Alat.tr.de pir.​=Alat.P​−Alat.p​
important
  • Aria laterală a unui trunchi de piramidă se poate determina și cu ajutorul formulei:
    • Alat.tr.de pir.=(PB+Pb)⋅atr2\mathcal{A}_{\text{lat.tr.de pir.}} = \frac{(\color{red}{P_B} + \color{blue}{P_b}) \cdot \color{green}{a_{tr}}}{2}Alat.tr.de pir.​=2(PB​+Pb​)⋅atr​​
    • unde PBP_BPB​ este perimetrul bazei mari, PbP_bPb​ este perimetrul bazei mici, iar atra_{tr}atr​ apotema trunchiului de piramidă.

Apotema trunchiului de piramidă regulată este înălțimea unei fețe laterale a trunchiului de piramidă.

important
  • Aria totală a trunchiului de piramidă regulată este suma dintre aria laterală a trunchiului de piramidă și ariile celor două baze.
    • Atot.tr.de pir.=Alat.tr.de pir.+AB+Ab\mathcal{A}_{\text{tot.tr.de pir.}} = \mathcal{A}_{\text{lat.tr.de pir.}} + \color{red}{\mathcal{A}_B} + \color{blue}{\mathcal{A}_b}Atot.tr.de pir.​=Alat.tr.de pir.​+AB​+Ab​
    • unde AB\mathcal{A}_BAB​ este aria bazei mari și Ab\mathcal{A}_bAb​ este aria bazei mici.
important
  • Volumul unui trunchi de piramidă regulată se poate determina ca diferență între volumul piramidei mari și volumul piramidei mici.
    • Vtr.de pir.=VP−VpV_{\text{tr.de pir.}} = \color{red}{V_P} - \color{blue}{V_p}Vtr.de pir.​=VP​−Vp​
    • unde VPV_PVP​ este volumul piramidei mari și VpV_pVp​ este volumul piramidei mici.
important
  • Volumul unui trunchi de piramidă se poate determina și cu formula:
    • Vtr.de pir.=h3(AB+Ab+AB⋅Ab)V_{\text{tr.de pir.}} = \frac{h}{3}(\color{red}{\mathcal{A}_B} + \color{blue}{\mathcal{A}_b} + \sqrt{\color{red}{\mathcal{A}_B} \cdot \color{blue}{\mathcal{A}_b}})Vtr.de pir.​=3h​(AB​+Ab​+AB​⋅Ab​​)
    • unde AB\mathcal{A}_BAB​ este aria bazei mari, Ab\mathcal{A}_bAb​ este aria bazei mici, iar hhh este înălțimea trunchiului de piramidă.
