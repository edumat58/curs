G8.2 - Trunchi de con circular drept

Aria laterală
  • Aria laterală a unui trunchi de con circular drept se poate determina ca diferență între aria laterală a conului mare și aria laterală a conului mic. Alat.tr.de con=Alat.Con−Alat.con\mathcal{A}_{lat.tr.de\ con} = \color{red}{\mathcal{A}_{lat.Con}} - \color{blue}{\mathcal{A}_{lat.con}}Alat.tr.de con​=Alat.Con​−Alat.con​
Formula ariei laterale
  • Aria laterală a unui trunchi de con circular drept se poate determina și cu formula: Alat.tr.de con=πG(R+r)\mathcal{A}_{lat.tr.de\ con} = \pi \color{green}{G}(\color{red}{R} + \color{blue}{r})Alat.tr.de con​=πG(R+r) unde R\color{red}{R}R este raza bazei mari, r\color{blue}{r}r este raza bazei mici, iar G\color{green}{G}G este generatoarea trunchiului de con, BB′BB'BB′.
Aria totală
  • Aria totală a trunchiului de con circular drept este suma dintre aria laterală a trunchiului de con circular drept și ariile celor două baze. Atot.tr.de con=πG(R+r)+πR2+πr2\mathcal{A}_{tot.tr.de\ con} = \pi \color{green}{G}(\color{red}{R} + \color{blue}{r}) + \pi \color{red}{R}^2 + \pi \color{blue}{r}^2Atot.tr.de con​=πG(R+r)+πR2+πr2
Volumul
  • Volumul unui trunchi de con circular drept se poate determina ca diferență între volumul conului mare și volumul conului mic. Vtr.de con=VCon−VconV_{tr.de\ con} = \color{red}{V_{Con}} - \color{blue}{V_{con}}Vtr.de con​=VCon​−Vcon​ unde VCon\color{red}{V_{Con}}VCon​ este volumul conului mare și Vcon\color{blue}{V_{con}}Vcon​ este volumul conului mic.
Formula volumului
  • Volumul unui trunchi de con circular drept se poate determina și cu formula: Vtr.de con=πh3(R2+r2+Rr)V_{tr.de\ con} = \frac{\pi \color{green}{h}}{3}(\color{red}{R}^2 + \color{blue}{r}^2 + \color{red}{R}\color{blue}{r})Vtr.de con​=3πh​(R2+r2+Rr) unde R\color{red}{R}R este raza bazei mari, r\color{blue}{r}r este raza bazei mici, iar h\color{green}{h}h este înălțimea trunchiului de con.

Notații​

De Reținut
  • RRR = raza bazei mari
  • rrr = raza bazei mici
  • GGG = generatoarea trunchiului
  • hhh = înălțimea trunchiului
