G5.2 - Cilindrul circular drept (arii și volum)

Elemente ale cilindrului​

  • raza: R=OA=O′A′R = OA = O'A'R=OA=O′A′
  • generatoarea: G=AA′=BB′G = AA' = BB'G=AA′=BB′
  • înălţimea: H=OO′H = OO'H=OO′ (pentru cilindrul drept, H=GH = GH=G)
Definitie

Cilindrul este un corp de rotaţie (se poate obţine prin rotirea unui dreptunghi în jurul unei laturi)

Secțiunea axială este dreptunghiul ABB′A′ABB'A'ABB′A′ de dimensiuni 2R2R2R și GGG.

Cilindrul se poate desfășura într-un plan, rezultând:

  • două cercuri congruente (baze)
  • un dreptunghi cu laturile 2πR2\pi R2πR și HHH

Formule​

Aria bazei​

de retinut

Ab=πR2A_b = \pi R^2Ab​=πR2

Aria laterală​

de retinut

Al=2πRGA_l = 2\pi R GAl​=2πRG

Aria totală​

de retinut

At=Al+2Ab=2πR(G+R)A_t = A_l + 2A_b = 2\pi R (G + R)At​=Al​+2Ab​=2πR(G+R)

Volumul​

de retinut

V=Ab⋅H=πR2GV = A_b \cdot H = \pi R^2 GV=Ab​⋅H=πR2G

Exemple​

Exemplul 1​

Fie un cilindru circular drept cu R=3,cmR = 3,cmR=3,cm și G=7,cmG = 7,cmG=7,cm. Aflați aria laterală, aria totală și volumul.

Rezolvare​

Aria bazei: Ab=πR2=π⋅32=9π,cm2A_b = \pi R^2 = \pi \cdot 3^2 = 9\pi,cm^2Ab​=πR2=π⋅32=9π,cm2

Aria laterală: Al=2πRG=2π⋅3⋅7=42π,cm2A_l = 2\pi R G = 2\pi \cdot 3 \cdot 7 = 42\pi,cm^2Al​=2πRG=2π⋅3⋅7=42π,cm2

Aria totală: At=Al+2Ab=42π+18π=60π,cm2A_t = A_l + 2A_b = 42\pi + 18\pi = 60\pi,cm^2At​=Al​+2Ab​=42π+18π=60π,cm2

Volumul: V=πR2G=π⋅32⋅7=63π,cm3V = \pi R^2 G = \pi \cdot 3^2 \cdot 7 = 63\pi,cm^3V=πR2G=π⋅32⋅7=63π,cm3

Exemplul 2​

Un cilindru are aria laterală 66π,cm266\pi,cm^266π,cm2 și volumul 363π,cm3363\pi,cm^3363π,cm3. Determinați aria secțiunii axiale.

Rezolvare​

Din aria laterală: Al=2πRGA_l = 2\pi R GAl​=2πRG 66π=2πRG ⇒RG=33(1)66\pi = 2\pi R G \ \Rightarrow RG = 33 \quad (1)66π=2πRG ⇒RG=33(1)

Din volum: V=πR2GV = \pi R^2 GV=πR2G 363π=πR2G363\pi = \pi R^2 G363π=πR2G Substituim GGG din (1): 363=R2⋅33R363 = R^2 \cdot \frac{33}{R}363=R2⋅R33​ 363=33R⇒R=11cm363 = 33R \Rightarrow R = 11 cm363=33R⇒R=11cm (2)

Din (1) și (2): 11⋅G=33⇒G=3,cm11 \cdot G = 33 \Rightarrow G = 3,cm11⋅G=33⇒G=3,cm

Secțiunea axială este dreptunghiul ABB′A′ABB'A'ABB′A′ cu dimensiunile 2R2R2R și GGG: Asec=2R⋅G=22⋅3=66,cm2A_{sec} = 2R \cdot G = 22 \cdot 3 = 66,cm^2Asec​=2R⋅G=22⋅3=66,cm2
