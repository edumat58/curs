C6 – Raționalizarea numitorului folosind conjugata

Introducere​

DE CE RAȚIONALIZĂM NUMITORUL?

Raționalizarea numitorului înseamnă transformarea unei fracții care are un radical în numitor într-o fracție echivalentă fără radical în numitor. Acest lucru se face pentru a simplifica calculele și pentru a obține forme mai ușor de lucrat.

Pentru numitori care sunt sume sau diferențe de termeni cu radicali, folosim metoda conjugatului.

Conjugatul unui număr​

DEFINIȚIE

Conjugatul unui număr de forma a+bca + b\sqrt{c}a+bc​, unde aaa și bbb sunt numere raționale, iar ccc este un număr natural care nu este pătrat perfect, este a−bca - b\sqrt{c}a−bc​.

Similar, conjugatul lui a−bca - b\sqrt{c}a−bc​ este a+bca + b\sqrt{c}a+bc​.

Produsul unui număr cu conjugatul său este întotdeauna un număr rațional: (a+bc)(a−bc)=a2−b2c(a + b\sqrt{c})(a - b\sqrt{c}) = a^2 - b^2 c(a+bc​)(a−bc​)=a2−b2c.

Metoda de raționalizare pentru numitor de forma a + b√c​

METODĂ

Pentru a raționaliza un numitor de forma a+bca + b\sqrt{c}a+bc​, unde aaa și bbb sunt numere raționale, iar ccc este un număr natural care nu este pătrat perfect:

  1. Înmulțește numărătorul și numitorul cu conjugatul numitorului: a−bca - b\sqrt{c}a−bc​.
  2. Simplifică expresia rezultată folosind identitatea remarcabilă (a+bc)(a−bc)=a2−(bc)2=a2−b2c(a + b\sqrt{c})(a - b\sqrt{c}) = a^2 - (b\sqrt{c})^2 = a^2 - b^2 c(a+bc​)(a−bc​)=a2−(bc​)2=a2−b2c.

Formula generală:

pa+bc=p(a−bc)(a+bc)(a−bc)=p(a−bc)a2−b2c\dfrac{p}{a + b\sqrt{c}} = \dfrac{p(a - b\sqrt{c})}{(a + b\sqrt{c})(a - b\sqrt{c})} = \dfrac{p(a - b\sqrt{c})}{a^2 - b^2 c}a+bc​p​=(a+bc​)(a−bc​)p(a−bc​)​=a2−b2cp(a−bc​)​

Exemple​

  1. Raționalizează 12+3\dfrac{1}{2 + \sqrt{3}}2+3​1​.

    Înmulțim cu conjugatul 2−32 - \sqrt{3}2−3​:

    12+3⋅2−32−3=2−3(2+3)(2−3)=2−34−3=2−31=2−3\dfrac{1}{2 + \sqrt{3}} \cdot \dfrac{2 - \sqrt{3}}{2 - \sqrt{3}} = \dfrac{2 - \sqrt{3}}{(2 + \sqrt{3})(2 - \sqrt{3})} = \dfrac{2 - \sqrt{3}}{4 - 3} = \dfrac{2 - \sqrt{3}}{1} = 2 - \sqrt{3}2+3​1​⋅2−3​2−3​​=(2+3​)(2−3​)2−3​​=4−32−3​​=12−3​​=2−3​

  2. Raționalizează 53+22\dfrac{5}{3 + 2\sqrt{2}}3+22​5​.

    Înmulțim cu conjugatul 3−223 - 2\sqrt{2}3−22​:

    53+22⋅3−223−22=5(3−22)(3+22)(3−22)=5(3−22)9−8=5(3−22)1=15−102\dfrac{5}{3 + 2\sqrt{2}} \cdot \dfrac{3 - 2\sqrt{2}}{3 - 2\sqrt{2}} = \dfrac{5(3 - 2\sqrt{2})}{(3 + 2\sqrt{2})(3 - 2\sqrt{2})} = \dfrac{5(3 - 2\sqrt{2})}{9 - 8} = \dfrac{5(3 - 2\sqrt{2})}{1} = 15 - 10\sqrt{2}3+22​5​⋅3−22​3−22​​=(3+22​)(3−22​)5(3−22​)​=9−85(3−22​)​=15(3−22​)​=15−102​

  3. Raționalizează 51+5\dfrac{\sqrt{5}}{1 + \sqrt{5}}1+5​5​​.

    Înmulțim cu conjugatul 1−51 - \sqrt{5}1−5​:

    51+5⋅1−51−5=5(1−5)(1+5)(1−5)=5−51−5=5−5−4=5−54\dfrac{\sqrt{5}}{1 + \sqrt{5}} \cdot \dfrac{1 - \sqrt{5}}{1 - \sqrt{5}} = \dfrac{\sqrt{5}(1 - \sqrt{5})}{(1 + \sqrt{5})(1 - \sqrt{5})} = \dfrac{\sqrt{5} - 5}{1 - 5} = \dfrac{\sqrt{5} - 5}{-4} = \dfrac{5 - \sqrt{5}}{4}1+5​5​​⋅1−5​1−5​​=(1+5​)(1−5​)5​(1−5​)​=1−55​−5​=−45​−5​=45−5​​

Metoda de raționalizare pentru numitor de forma a - b√c​

METODĂ

Pentru un numitor de forma a−bca - b\sqrt{c}a−bc​, procedeul este similar. Înmulțim cu conjugatul a+bca + b\sqrt{c}a+bc​.

Formula generală:

pa−bc=p(a+bc)(a−bc)(a+bc)=p(a+bc)a2−b2c\dfrac{p}{a - b\sqrt{c}} = \dfrac{p(a + b\sqrt{c})}{(a - b\sqrt{c})(a + b\sqrt{c})} = \dfrac{p(a + b\sqrt{c})}{a^2 - b^2 c}a−bc​p​=(a−bc​)(a+bc​)p(a+bc​)​=a2−b2cp(a+bc​)​

Exemple​

  1. Raționalizează 12−3\dfrac{1}{2 - \sqrt{3}}2−3​1​.

    Înmulțim cu conjugatul 2+32 + \sqrt{3}2+3​:

    12−3⋅2+32+3=2+34−3=2+3\dfrac{1}{2 - \sqrt{3}} \cdot \dfrac{2 + \sqrt{3}}{2 + \sqrt{3}} = \dfrac{2 + \sqrt{3}}{4 - 3} = 2 + \sqrt{3}2−3​1​⋅2+3​2+3​​=4−32+3​​=2+3​

  2. Raționalizează 45−32\dfrac{4}{5 - 3\sqrt{2}}5−32​4​.

    Înmulțim cu conjugatul 5+325 + 3\sqrt{2}5+32​:

    45−32⋅5+325+32=4(5+32)25−18=4(5+32)7=20+1227\dfrac{4}{5 - 3\sqrt{2}} \cdot \dfrac{5 + 3\sqrt{2}}{5 + 3\sqrt{2}} = \dfrac{4(5 + 3\sqrt{2})}{25 - 18} = \dfrac{4(5 + 3\sqrt{2})}{7} = \dfrac{20 + 12\sqrt{2}}{7}5−32​4​⋅5+32​5+32​​=25−184(5+32​)​=74(5+32​)​=720+122​​

Proprietăți și observații​

PROPRIETĂȚI IMPORTANTE
  1. Conjugatul unui număr de forma a+bca + b\sqrt{c}a+bc​ este a−bca - b\sqrt{c}a−bc​, și invers.
  2. Produsul unui număr cu conjugatul său este întotdeauna un număr rațional: (a+bc)(a−bc)=a2−b2c(a + b\sqrt{c})(a - b\sqrt{c}) = a^2 - b^2 c(a+bc​)(a−bc​)=a2−b2c.
  3. Dacă numitorul conține doar un radical simplu (fără sumă sau diferență), folosim metodele anterioare (înmulțire cu radicalul).
  4. Raționalizarea ajută la compararea, adunarea sau înmulțirea fracțiilor cu radicali în numitor.

Exemplu de aplicare​

Calculează 11+2+11−2\dfrac{1}{1 + \sqrt{2}} + \dfrac{1}{1 - \sqrt{2}}1+2​1​+1−2​1​.

Raționalizăm fiecare fracție:

  • 11+2=1−2\dfrac{1}{1 + \sqrt{2}} = 1 - \sqrt{2}1+2​1​=1−2​
  • 11−2=1+2\dfrac{1}{1 - \sqrt{2}} = 1 + \sqrt{2}1−2​1​=1+2​

Suma: (1−2)+(1+2)=2(1 - \sqrt{2}) + (1 + \sqrt{2}) = 2(1−2​)+(1+2​)=2

Metoda de raționalizare pentru numitor de forma √a ± √b​

DEFINIȚIE

Pentru expresii de forma a±b\sqrt{a} \pm \sqrt{b}a​±b​, conjugatul este a∓b\sqrt{a} \mp \sqrt{b}a​∓b​.

Produsul: (a+b)(a−b)=a−b(\sqrt{a} + \sqrt{b})(\sqrt{a} - \sqrt{b}) = a - b(a​+b​)(a​−b​)=a−b.

METODĂ

Pentru a raționaliza un numitor de forma a+b\sqrt{a} + \sqrt{b}a​+b​ sau a−b\sqrt{a} - \sqrt{b}a​−b​:

  1. Înmulțește numărătorul și numitorul cu conjugatul: pentru a+b\sqrt{a} + \sqrt{b}a​+b​, conjugatul este a−b\sqrt{a} - \sqrt{b}a​−b​, și invers.
  2. Simplifică folosind identitatea (a+b)(a−b)=a−b(\sqrt{a} + \sqrt{b})(\sqrt{a} - \sqrt{b}) = a - b(a​+b​)(a​−b​)=a−b.

Formula generală:

pa+b=p(a−b)(a+b)(a−b)=p(a−b)a−b\dfrac{p}{\sqrt{a} + \sqrt{b}} = \dfrac{p(\sqrt{a} - \sqrt{b})}{(\sqrt{a} + \sqrt{b})(\sqrt{a} - \sqrt{b})} = \dfrac{p(\sqrt{a} - \sqrt{b})}{a - b}a​+b​p​=(a​+b​)(a​−b​)p(a​−b​)​=a−bp(a​−b​)​

Exemple​

  1. Raționalizează 13+2\dfrac{1}{\sqrt{3} + \sqrt{2}}3​+2​1​.

    Înmulțim cu conjugatul 3−2\sqrt{3} - \sqrt{2}3​−2​:

    13+2⋅3−23−2=3−23−2=3−2\dfrac{1}{\sqrt{3} + \sqrt{2}} \cdot \dfrac{\sqrt{3} - \sqrt{2}}{\sqrt{3} - \sqrt{2}} = \dfrac{\sqrt{3} - \sqrt{2}}{3 - 2} = \sqrt{3} - \sqrt{2}3​+2​1​⋅3​−2​3​−2​​=3−23​−2​​=3​−2​

  2. Raționalizează 55−3\dfrac{5}{\sqrt{5} - \sqrt{3}}5​−3​5​.

    Înmulțim cu conjugatul 5+3\sqrt{5} + \sqrt{3}5​+3​:

    55−3⋅5+35+3=5(5+3)5−3=5(5+3)2\dfrac{5}{\sqrt{5} - \sqrt{3}} \cdot \dfrac{\sqrt{5} + \sqrt{3}}{\sqrt{5} + \sqrt{3}} = \dfrac{5(\sqrt{5} + \sqrt{3})}{5 - 3} = \dfrac{5(\sqrt{5} + \sqrt{3})}{2}5​−3​5​⋅5​+3​5​+3​​=5−35(5​+3​)​=25(5​+3​)​
