G3 - Distanţa dintre două puncte

Ce este distanța dintre două puncte?​

Definiție

Distanța dintre două puncte AAA și BBB este egală cu lungimea segmentului [AB][AB][AB] care le unește.

Distanța dintre AAA și BBB se notează la fel cu lungimea segmentului:

ABsaud(A,B)AB \quad \text{sau} \quad d(A, B)ABsaud(A,B)
Observație
  • Distanța este întotdeauna un număr pozitiv (sau 000).
  • Distanța de la un punct la el însuși este 000:   d(A,A)=0\;d(A, A) = 0d(A,A)=0.
  • Distanța nu depinde de ordinea punctelor:   AB=BA\;AB = BAAB=BA.

Măsurarea distanței cu rigla​

Pentru a măsura distanța dintre două puncte de pe o foaie, folosim rigla gradată:

Așezăm rigla astfel încât punctul AAA să fie în dreptul gradației 222, iar punctul BBB în dreptul gradației 777. Distanța este diferența dintre cele două citiri:

AB=7−2=5 cmAB = 7 - 2 = 5\,cmAB=7−2=5cm
Sfat practic

Putem așeza rigla și cu gradația 000 exact în punctul AAA — atunci distanța se citește direct în dreptul punctului BBB.

Distanțe pe o dreaptă (puncte coliniare)​

Când trei puncte sunt coliniare (se află pe aceeași dreaptă), distanțele dintre ele se pot aduna sau scădea.

Considerăm punctul CCC situat între AAA și BBB:

Regula adunării distanțelor

Dacă punctul CCC se află între AAA și BBB, atunci:

AC+CB=ABAC + CB = ABAC+CB=AB

De aici putem afla oricare dintre cele trei distanțe, dacă le cunoaștem pe celelalte două:

AC=AB−CBCB=AB−ACAC = AB - CB \qquad CB = AB - ACAC=AB−CBCB=AB−AC
Exemplu rezolvat

Punctele AAA, CCC, BBB sunt coliniare, în această ordine, cu AC=4 cmAC = 4\,cmAC=4cm și CB=9 cmCB = 9\,cmCB=9cm. Cât este ABABAB?

Cum CCC este între AAA și BBB:

AB=AC+CB=4+9=13 cmAB = AC + CB = 4 + 9 = 13\,cmAB=AC+CB=4+9=13cm
Exemplu rezolvat

Punctele AAA, CCC, BBB sunt coliniare, în această ordine, cu AB=20 cmAB = 20\,cmAB=20cm și AC=12 cmAC = 12\,cmAC=12cm. Cât este CBCBCB?

CB=AB−AC=20−12=8 cmCB = AB - AC = 20 - 12 = 8\,cmCB=AB−AC=20−12=8cm
Atenție la ordinea punctelor!

Regula AC+CB=ABAC + CB = ABAC+CB=AB este valabilă doar dacă CCC se află între AAA și BBB. Dacă punctele sunt în altă ordine pe dreaptă, relația se schimbă.

De exemplu, dacă ordinea este A, B, CA,\,B,\,CA,B,C (deci BBB este între AAA și CCC), atunci AB+BC=ACAB + BC = ACAB+BC=AC.

Calcule cu distanțe​

Exemplu rezolvat

Pe o dreaptă, în ordinea A, M, BA,\, M,\, BA,M,B, avem AM=2xAM = 2xAM=2x și MB=x+5MB = x + 5MB=x+5 (în cm). Știind că AB=17 cmAB = 17\,cmAB=17cm, află xxx.

Cum MMM este între AAA și BBB:

AM+MB=ABAM + MB = ABAM+MB=AB2x+(x+5)=172x + (x + 5) = 172x+(x+5)=173x+5=17  ⇒  3x=12  ⇒  x=43x + 5 = 17 \;\Rightarrow\; 3x = 12 \;\Rightarrow\; x = 43x+5=17⇒3x=12⇒x=4

Verificare: AM=2⋅4=8 cmAM = 2 \cdot 4 = 8\,cmAM=2⋅4=8cm, MB=4+5=9 cmMB = 4 + 5 = 9\,cmMB=4+5=9cm, iar 8+9=17 cm8 + 9 = 17\,cm8+9=17cm. ✓

Exemplu rezolvat

Pe o dreaptă se află punctele AAA, BBB, CCC în această ordine, cu AB=6 cmAB = 6\,cmAB=6cm și BC=10 cmBC = 10\,cmBC=10cm. Cât este distanța ACACAC?

Cum BBB este între AAA și CCC:

AC=AB+BC=6+10=16 cmAC = AB + BC = 6 + 10 = 16\,cmAC=AB+BC=6+10=16cm
