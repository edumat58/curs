C9 - Proprietăți puteri cu exponent natural

Introducere​

Ce este o putere?

Puterea unui număr reprezintă produsul acelui număr cu el însuși de mai multe ori. Dacă aaa este un număr și n≥1n \ge 1n≥1, atunci:

an=a×a×⋯×a⏟n factoria^n = \underbrace{a \times a \times \dots \times a}_{n\ \text{factori}}an=n factoria×a×⋯×a​​

Se citește „aaa la puterea nnn” sau „aaa la exponent nnn”.

Exemple:

  • 25=2×2×2×2×2=322^5 = 2 \times 2 \times 2 \times 2 \times 2 = 3225=2×2×2×2×2=32

  • 20001=20002000^1 = 200020001=2000

  • (−3)2=(−3)×(−3)=9(-3)^2 = (-3) \times (-3) = 9(−3)2=(−3)×(−3)=9

  • (−3)3=(−3)×(−3)×(−3)=−27(-3)^3 = (-3) \times (-3) \times (-3) = -27(−3)3=(−3)×(−3)×(−3)=−27

  • (23)3=2×2×23×3×3=827\left(\dfrac{2}{3}\right)^3 = \dfrac{2 \times 2 \times 2}{3 \times 3 \times 3} = \dfrac{8}{27}(32​)3=3×3×32×2×2​=278​

  • 03=00^3 = 003=0

Observații

  • a2a^2a2 se citește „aaa la pătrat”

  • a3a^3a3 se citește „aaa la cub”

  • Atenție: 23=82^3 = 823=8, pe când 3×2=63 \times 2 = 63×2=6.

1. Puteri ale unui număr​

Exponent pozitiv​

Dacă n≥1n \ge 1n≥1 și aaa este un număr real,

an=a×a×⋯×aa^n = a \times a \times \dots \times aan=a×a×⋯×a

(cu nnn factori egali cu aaa).

Produsul al două puteri cu aceeași bază​

Pentru aaa număr real și m,n≥1m, n \ge 1m,n≥1:

am×an=am+na^m \times a^n = a^{m+n}am×an=am+n

Exemple:

  • 23×24=23+4=272^3 \times 2^4 = 2^{3+4} = 2^723×24=23+4=27
  • 52×51=535^2 \times 5^1 = 5^{3}52×51=53
  • 83×82×84=83+2+4=898^3 \times 8^2 \times 8^4 = 8^{3+2+4} = 8^{9}83×82×84=83+2+4=89
Atenție

Dacă bazele sunt diferite, regula nu se aplică: 52×435^2 \times 4^352×43 nu se poate scrie ca o singură putere.

Nici o suma de puteri nu se reduce: 36+323^6 + 3^236+32 nu este o singură putere.

Puterea cu exponent zero​

De Retinut

Observăm că a0=1a^0 = 1a0=1 pentru orice număr real aaa:

a0×a4=a0+4=a4⇒a0=1a^0 \times a^4 = a^{0+4} = a^4 \quad\Rightarrow\quad a^0 = 1a0×a4=a0+4=a4⇒a0=1

În special, 00=10^0 = 100=1 (convenție).

Puterea unei puteri​

Pentru aaa număr real și m,n≥1m, n \ge 1m,n≥1:

(am)n=am×n\bigl(a^m\bigr)^n = a^{m \times n}(am)n=am×n

Exemple:

  • (23)2=23×2=26(2^3)^2 = 2^{3 \times 2} = 2^6(23)2=23×2=26
  • (76)3=76×3=718(7^6)^3 = 7^{6 \times 3} = 7^{18}(76)3=76×3=718

Câtul a două puteri cu aceeași bază​

Pentru a≠0a \neq 0a=0 și m,nm, nm,n naturali:

aman=am−n\dfrac{a^m}{a^n} = a^{m-n}anam​=am−n

Exemple:

  • 2522=25−2=23\dfrac{2^5}{2^2} = 2^{5-2} = 2^32225​=25−2=23

  • 3634=36−4=32\dfrac{3^6}{3^4} = 3^{6-4} = 3^{2}3436​=36−4=32

Puterea unui produs și a unui cât​

Pentru a,b≠0a, b \neq 0a,b=0 și nnn întreg:

(a×b)n=an×bn,(ab)n=anbn(a \times b)^n = a^n \times b^n, \qquad \left(\dfrac{a}{b}\right)^n = \dfrac{a^n}{b^n}(a×b)n=an×bn,(ba​)n=bnan​

Exemple:

  • (2×3)4=24×34(2 \times 3)^4 = 2^4 \times 3^4(2×3)4=24×34
  • (25)3=2353\left(\dfrac{2}{5}\right)^3 = \dfrac{2^3}{5^3}(52​)3=5323​
  • 43×73=(4×7)3=2834^3 \times 7^3 = (4 \times 7)^3 = 28^343×73=(4×7)3=283

2. Puterile lui 10​

Exemple de bază​

  • 103=10×10×10=1 00010^3 = 10 \times 10 \times 10 = 1\,000103=10×10×10=1000
  • 10−2=1102=0,0110^{-2} = \dfrac{1}{10^2} = 0{,}0110−2=1021​=0,01
Proprietate

Pentru n≥1n \ge 1n≥1:

  • 10n10^n10n este numărul 1 urmat de nnn zerouri.
  • 10−n=110n10^{-n} = \dfrac{1}{10^n}10−n=10n1​ reprezintă numărul cu nnn cifre după virgulă, de forma 0,00…010,00\dots 010,00…01.

Exemple:

  • 105=100 00010^5 = 100\,000105=100000
  • 10−4=0,000110^{-4} = 0{,}000110−4=0,0001
  • 100=110^0 = 1100=1
  • 101=1010^1 = 10101=10
  • 10−1=0,110^{-1} = 0{,}110−1=0,1

Reguli de calcul​

De Reținut
10m×10n=10m+n,10m10n=10m−n,(10m)n=10m×n10^m \times 10^n = 10^{m+n}, \quad \dfrac{10^m}{10^n} = 10^{m-n}, \quad \bigl(10^m\bigr)^n = 10^{m \times n}10m×10n=10m+n,10n10m​=10m−n,(10m)n=10m×n

Exemple:

  • 103×104=10710^3 \times 10^4 = 10^{7}103×104=107

  • 107103=104\dfrac{10^7}{10^3} = 10^{4}103107​=104

  • (105)2=1010\bigl(10^5\bigr)^2 = 10^{10}(105)2=1010

Deplasarea virgulei​

  • Înmulțirea unui număr zecimal cu 10n10^n10n deplasează virgula nnn ranguri spre dreapta.
  • Înmulțirea cu 10−n10^{-n}10−n deplasează virgula nnn ranguri spre stânga.

Exemple:

  • 25,1×105=2 510 00025{,}1 \times 10^5 = 2\,510\,00025,1×105=2510000
  • 25,1×10−1=25,1:10=2,5125{,}1 \times 10^{-1} = 25,1 : 10 = 2{,}5125,1×10−1=25,1:10=2,51

3. Scrierea (notația) științifică​

Definiție

Scrierea științifică a unui număr real este forma:

a×10na \times 10^na×10n

unde aaa este un număr zecimal cu un singur cifru nenul înainte de virgulă, iar nnn este un număr întreg.

Exemple:

  • A=8,56×107A = 8{,}56 \times 10^7A=8,56×107 este scris corect în notație științifică.
  • B=0,45×102B = 0{,}45 \times 10^{2}B=0,45×102 nu este scris corect (cifra înainte de virgulă este nulă).
  • C=9,1×53C = 9{,}1 \times 5^3C=9,1×53 nu este scris corect (al doilea factor nu este o putere a lui 10).

Compararea numerelor scrise științific​

Pentru A=a×10mA = a \times 10^mA=a×10m și B=b×10nB = b \times 10^nB=b×10n:

  1. Comparăm mai întâi exponenții mmm și nnn.
  2. Dacă sunt egali, comparăm valorile aaa și bbb.

Exemple:

  • 6,04×105<2,03×1076{,}04 \times 10^5 < 2{,}03 \times 10^76,04×105<2,03×107 (deoarece 5<75<75<7)
  • 4,51×107<6,7×1074{,}51 \times 10^7 < 6{,}7 \times 10^74,51×107<6,7×107 (exponenți egali, comparăm coeficienții)
