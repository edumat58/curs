# C7.1 – Ilustrarea fracţiilor echiunitare, subunitare și supraunitare

---

## Introducere

:::tip Ce sunt fracțiile?
Fracțiile reprezintă părți dintr-un întreg. În funcție de mărimea lor față de unitate, le clasificăm în:

- **Fracții echiunitare** (unitare): numitorul este egal cu numărătorul. 

Exemple: $ \dfrac{1}{1} $, $ \dfrac{2}{2} $

- **Fracții subunitare** (proprii): mai mici decât $1$. 

Exemple: $ \dfrac{1}{2} $, $ \dfrac{3}{4} $

- **Fracții supraunitare** (improprii): mai mari decât $1$. 

Exemple: $ \dfrac{3}{2} $, $ \dfrac{5}{3} $
:::
