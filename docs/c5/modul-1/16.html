C14 - Media aritmetică

Media aritmetică a două numere naturale​

Definitie 1

Media aritmetică a două numere naturale este un concept pe care îl întâlnim în numeroase activități cotidiene.

Media aritmetică a două numere naturale aaa și bbb este numărul obținut prin împărțirea la 2 a sumei lor:

ma=a+b2ma = \frac{a + b}{2}ma=2a+b​

Exemplu:

O gospodină cumpără de la piață 1 kg de prune pentru care plătește 2 lei și 1 kg de struguri pentru care plătește 4 lei.
Cât plătește gospodina pentru un kilogram de fructe?

Rezolvare:

  • pentru 2 kg de fructe, gospodina plătește: 2+4=62 + 4 = 62+4=6 lei
  • pentru 1 kg de fructe, gospodina plătește: 6:2=36 : 2 = 36:2=3 lei

Deci, pentru un kilogram de fructe gospodina plătește:

(2+4):2=3 (lei)(2 + 4) : 2 = 3 \text{ (lei)}(2+4):2=3 (lei)

Media aritmetică a prețului unui kilogram de fructe este de 3 lei.

  1. Media aritmetică a numerelor 7 și 11 este numărul:
ma=7+112=182=9ma = \frac{7 + 11}{2} = \frac{18}{2} = 9ma=27+11​=218​=9
  1. Media aritmetică a numerelor 12 și 28 este numărul:
ma=12+282=402=20ma = \frac{12 + 28}{2} = \frac{40}{2} = 20ma=212+28​=240​=20
Observație

Media aritmetică a două numere naturale nu este totdeauna număr natural.

Exemplu:

Media aritmetică a numerelor 5 și 8 este:

ma=5+82=132ma = \frac{5 + 8}{2} = \frac{13}{2}ma=25+8​=213​

care nu este număr natural.

Media aritmetică a trei numere naturale​

Definitie 2

Media aritmetică a trei numere naturale aaa, bbb și ccc este numărul obținut prin împărțirea la 3 a sumei lor:

ma=a+b+c3ma = \frac{a + b + c}{3}ma=3a+b+c​

Exemplu: Media aritmetică a numerelor 6, 15 și 24 este numărul:

ma=6+15+243=453=15ma = \frac{6 + 15 + 24}{3} = \frac{45}{3} = 15ma=36+15+24​=345​=15
Observație

Media aritmetică a nnn numere naturale distincte este mai mare decât cel mai mic dintre numere și mai mică decât cel mai mare dintre numere.

Exemplu: Media aritmetică a numerelor 2, 18, 21 și 31 este:

ma=2+18+21+314=724=18ma = \frac{2 + 18 + 21 + 31}{4} = \frac{72}{4} = 18ma=42+18+21+31​=472​=18

iar:

min⁡{2,18,21,31}=2șimax⁡{2,18,21,31}=31\min\{2, 18, 21, 31\} = 2 \quad \text{și} \quad \max\{2, 18, 21, 31\} = 31min{2,18,21,31}=2șimax{2,18,21,31}=31

Se verifică relația:

2≤18≤312 \le 18 \le 312≤18≤31

Media aritmetică a n numere naturale​

Definitie 3

Fiind date nnn numere naturale notate a1,a2,a3,…,ana_1, a_2, a_3, \ldots, a_na1​,a2​,a3​,…,an​,
se numește media aritmetică numărul obținut prin împărțirea la nnn a sumei lor:

ma=a1+a2+a3+…+annma = \frac{a_1 + a_2 + a_3 + \ldots + a_n}{n}ma=na1​+a2​+a3​+…+an​​

Exemplu: La sfârșitul primului semestru Maria are la istorie notele: 9, 10, 7, 10, 9.
Media aritmetică a notelor ei la istorie este egală cu:

ma=9+10+7+10+95=455=9ma = \frac{9 + 10 + 7 + 10 + 9}{5} = \frac{45}{5} = 9ma=59+10+7+10+9​=545​=9
