G7 - Tipuri de unghiuri. Măsura unghiurilor

Ce este un unghi?​

Definiție

Un unghi este figura geometrică formată din două semidrepte care au aceeași origine.

Cele două semidrepte se numesc laturile unghiului, iar originea lor comună se numește vârful unghiului.

Unghiul din figură se notează în mai multe feluri:

∠AOB,∠BOA,∠O,α\angle AOB,\quad \angle BOA,\quad \angle O,\quad \alpha∠AOB,∠BOA,∠O,α
Observație

Când notăm un unghi cu trei litere, litera din mijloc reprezintă întotdeauna vârful unghiului.

Măsura unghiurilor​

Măsura unui unghi se exprimă în grade, notate cu simbolul ∘^\circ∘.

De reținut

Un unghi complet (o rotație întreagă) are măsura de 360∘360^\circ360∘.

1 rotație completa˘=360∘\text{1 rotație completă} = 360^\circ1 rotație completa˘=360∘

Gradul se împarte la rândul lui în unități mai mici:

1∘=60′(minute)1^\circ = 60' \quad (\text{minute})1∘=60′(minute) 1′=60′′(secunde)1' = 60'' \quad (\text{secunde})1′=60′′(secunde)

Așadar:

1∘=60′=3600′′1^\circ = 60' = 3600''1∘=60′=3600′′

Tipuri de unghiuri după măsură​

Clasificăm unghiurile în funcție de măsura lor astfel:

Tabelul de mai jos rezumă clasificarea:

Tipul unghiuluiMăsura
Unghi nul0∘0^\circ0∘
Unghi ascuțit0∘<α<90∘0^\circ < \alpha < 90^\circ0∘<α<90∘
Unghi dreptα=90∘\alpha = 90^\circα=90∘
Unghi obtuz90∘<α<180∘90^\circ < \alpha < 180^\circ90∘<α<180∘
Unghi alungitα=180∘\alpha = 180^\circα=180∘
Unghi completα=360∘\alpha = 360^\circα=360∘
Observație

Un unghi drept se marchează de obicei cu un mic pătrat în vârf, în loc de un arc.

Perechi remarcabile de unghiuri​

Unghiuri complementare​

Definiție

Două unghiuri sunt complementare dacă suma măsurilor lor este 90∘90^\circ90∘.

α+β=90∘\alpha + \beta = 90^\circα+β=90∘

Fiecare dintre cele două unghiuri se numește complementul celuilalt.

Unghiuri suplementare​

Definiție

Două unghiuri sunt suplementare dacă suma măsurilor lor este 180∘180^\circ180∘.

α+β=180∘\alpha + \beta = 180^\circα+β=180∘

Fiecare dintre cele două unghiuri se numește suplementul celuilalt.

Unghiuri opuse la vârf​

Definiție

Două unghiuri sunt opuse la vârf dacă laturile unuia sunt prelungirile laturilor celuilalt.

Proprietate

Două unghiuri opuse la vârf sunt congruente (au aceeași măsură).

În figură, unghiurile notate cu α\alphaα sunt opuse la vârf, deci sunt egale. La fel, cele notate cu β\betaβ.

Calcule cu unghiuri​

Adunarea și scăderea măsurilor​

Când lucrăm cu grade, minute și secunde, ținem cont că:

1∘=60′,1′=60′′1^\circ = 60', \qquad 1' = 60''1∘=60′,1′=60′′
Exemplu rezolvat

Calculează:

35∘40′+28∘30′35^\circ 40' + 28^\circ 30'35∘40′+28∘30′

Rezolvare:

Adunăm separat gradele și minutele:

(35∘+28∘)+(40′+30′)=63∘+70′(35^\circ + 28^\circ) + (40' + 30') = 63^\circ + 70'(35∘+28∘)+(40′+30′)=63∘+70′

Cum 70′=60′+10′=1∘10′70' = 60' + 10' = 1^\circ 10'70′=60′+10′=1∘10′, transformăm:

63∘+1∘10′=64∘10′63^\circ + 1^\circ 10' = 64^\circ 10'63∘+1∘10′=64∘10′

Aflarea complementului și a suplementului​

Exemplu rezolvat

Află complementul și suplementul unghiului de 35∘35^\circ35∘.

Complement:

90∘−35∘=55∘90^\circ - 35^\circ = 55^\circ90∘−35∘=55∘

Suplement:

180∘−35∘=145∘180^\circ - 35^\circ = 145^\circ180∘−35∘=145∘
