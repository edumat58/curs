08 - Ecuații și inecuații în ℤ

Definiții teoretice (26.03.2025)​

  1. Definiție: Se numește ecuație orice egalitate a două expresii algebrice care conțin una sau mai multe valori necunoscute (x).

  2. Definiție: Inecuatiile sunt inegalități adevărate doar pentru anumite valori atribuite literelor: ax+b<cax + b < cax+b<c, >>>, ≤\leq≤, ≥\geq≥

Suma lui Gauss​

1+2+3+⋯+n=n(n+1)21 + 2 + 3 + \dots + n = \frac{n(n + 1)}{2}1+2+3+⋯+n=2n(n+1)​

Exemplu:

x(1+2+⋯+100)=−5050⇒x⋅100⋅1012=−5050x(1 + 2 + \dots + 100) = -5050 \Rightarrow x \cdot \frac{100 \cdot 101}{2} = -5050x(1+2+⋯+100)=−5050⇒x⋅2100⋅101​=−5050 x⋅5050=−5050⇒x=−1x \cdot 5050 = -5050 \Rightarrow x = -1x⋅5050=−5050⇒x=−1
