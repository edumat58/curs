C6.1 - Mărimi direct proporționale

Definiție​

Definitie

Două mărimi se numesc direct proporționale dacă atunci când una crește (sau se micșorează) de un număr de ori, și cealaltă crește (sau se micșorează) de același număr de ori.

Exemplu

Cantitatea de mere și cantitatea de suc obținut:

Cantitatea de mere (kg)Cantitatea de suc (l)
104
3012
52

Observăm că:

  • 1030=412=42=13\dfrac{10}{30} = \dfrac{4}{12} = \dfrac{4}{2} = \dfrac{1}{3}3010​=124​=24​=31​

  • 104=3012=52\dfrac{10}{4} = \dfrac{30}{12} = \dfrac{5}{2}410​=1230​=25​

Proprietăți​

Proprietatea 1

Raportul a două valori din prima mărime este egal cu raportul valorilor corespunzătoare din cealaltă mărime.

a1a2=b1b2\frac{a_1}{a_2} = \frac{b_1}{b_2}a2​a1​​=b2​b1​​
Proprietatea 2

Raportul valorilor corespunzătoare din cele două mărimi este constant.

a1b1=a2b2=⋯=k\frac{a_1}{b_1} = \frac{a_2}{b_2} = \cdots = kb1​a1​​=b2​a2​​=⋯=k

Mulțimi de numere​

Definitie

Fiind date două mulțimi
A={a1,a2,…,an},B={b1,b2,…,bn},A=\{a_1,a_2,\dots,a_n\},\quad B=\{b_1,b_2,\dots,b_n\},A={a1​,a2​,…,an​},B={b1​,b2​,…,bn​},
vom spune că între elementele acestora există o dependență direct proporțională (sunt direct proporționale) dacă

a1b1=a2b2=⋯=anbn.\frac{a_1}{b_1} = \frac{a_2}{b_2} = \cdots = \frac{a_n}{b_n}.b1​a1​​=b2​a2​​=⋯=bn​an​​.

Probleme rezolvate​

Problema 1
  1. 12 kg de mere costă 34 lei. Câte kilograme de mere putem cumpăra cu 18 lei?

Rezolvare

1234=x18  ⟹  x=12⋅1834=6,35 kg\frac{12}{34} = \frac{x}{18} \implies x = \frac{12\cdot18}{34} = 6{,}35\text{ kg}3412​=18x​⟹x=3412⋅18​=6,35 kg
Problema 2
  1. Fie xxx, yyy, zzz trei numere direct proporționale cu numerele 3, 5, 7. Aflați xxx, yyy, zzz dacă x+y+z=75x+y+z=75x+y+z=75.

Rezolvare
x:y:z=3:5:7x : y : z = 3 : 5 : 7x:y:z=3:5:7

x3=y5=z7=k\frac{x}{3} = \frac{y}{5} = \frac{z}{7} = k 3x​=5y​=7z​=k

3k+5k+7k=75  ⟹  15k=75  ⟹  k=53k + 5k + 7k = 75 \implies 15k = 75 \implies k = 53k+5k+7k=75⟹15k=75⟹k=5

Deci x=3⋅5=15, y=5⋅5=25, z=7⋅5=35x = 3\cdot5 = 15,\ y = 5\cdot5 = 25,\ z = 7\cdot5 = 35x=3⋅5=15, y=5⋅5=25, z=7⋅5=35

Problema 3
  1. Împărțiți numărul 450 în părți direct proporționale cu numerele 2, 3, 5, 8.

Rezolvare
x:y:z:t=2:3:5:8x : y : z : t = 2 : 3 : 5 : 8x:y:z:t=2:3:5:8

x2=y3=z5=t8=k\frac{x}{2} = \frac{y}{3} = \frac{z}{5} = \frac{t}{8} = k 2x​=3y​=5z​=8t​=k

2k+3k+5k+8k=450  ⟹  18k=450  ⟹  k=252k + 3k + 5k + 8k = 450 \implies 18k = 450 \implies k = 252k+3k+5k+8k=450⟹18k=450⟹k=25

Deci x=50, y=75, z=125, t=200x=50,\ y=75,\ z=125,\ t=200x=50, y=75, z=125, t=200.

Problema 4
  1. Dacă numerele xxx, yyy, zzz sunt direct proporționale cu numerele 9, 15, 11, aflați câte procente reprezintă numărul cel mai mare din suma celorlalte două.

Rezolvare
x:y:z=9k:15k:11kx : y : z = 9k : 15k : 11kx:y:z=9k:15k:11k

x9=y15=z11=k\frac{x}{9} = \frac{y}{15} = \frac{z}{11} = k9x​=15y​=11z​=k

Numărul cel mai mare este 15k15k15k. Suma celorlalte două este 9k+11k=20k9k + 11k = 20k9k+11k=20k.
Procentul cerut:

15k20k⋅100%=75%\frac{15k}{20k}\cdot100\% = 75\%20k15k​⋅100%=75%
