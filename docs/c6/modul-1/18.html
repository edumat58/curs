C9 - Relații și operații între mulțimi

Relații între mulțimi​

Două mulțimi AAA și BBB sunt egale dacă au aceleași elemente.
Notăm:
A=B.A = B.A=B.

Două mulțimi AAA și BBB, care nu au aceleași elemente, nu sunt egale, chiar dacă au același număr de elemente.
Notăm:
A≠B.A \ne B.A=B.

Submulțimi​

Definite 1

O mulțime AAA este submulțime a mulțimii BBB (spunem că mulțimea AAA este inclusă în mulțimea BBB)
dacă fiecare element xxx din AAA este în același timp și element al mulțimii BBB.
Notăm:
A⊂B.A \subset B.A⊂B.

Dacă există cel puțin un element al mulțimii AAA care nu este element al mulțimii BBB,
atunci AAA nu este submulțime a lui BBB și notăm:
A⊄B.A \not\subset B.A⊂B.

Mulțimea vidă și submulțimile improprii​

Definitie 2

Mulțimea vidă este submulțime a oricărei mulțimi.

Orice mulțime AAA are submulțimile AAA și ∅\varnothing∅, numite submulțimi improprii.

Orice submulțime a lui AAA, diferită de ∅\varnothing∅ și de AAA, dacă există, se numește submulțime proprie a lui AAA.

Proprietate

Dacă A⊂BA \subset BA⊂B și B⊂AB \subset AB⊂A, atunci A=BA = BA=B

Mulțimea părților​

Definitie 3

Mulțimea tuturor submulțimilor unei mulțimi MMM formează mulțimea părților mulțimii MMM și se notează:

P(M)P(M)P(M)

Exemple:

Exemplul 1

Considerând diagramele de mai jos, avem:

B⊂AB \subset AB⊂A

Exemplul 2

Fie:

M={4,5,6,7,8},N={4,7},P={7,8,9}M = \{4, 5, 6, 7, 8\}, \quad N = \{4, 7\}, \quad P = \{7, 8, 9\}M={4,5,6,7,8},N={4,7},P={7,8,9}

Avem:

N⊂MșiP⊄M (deoarece 9∉M)N \subset M \quad \text{și} \quad P \not\subset M \text{ (deoarece } 9 \notin M)N⊂MșiP⊂M (deoarece 9∈/M)

Exemplul 3

Fie:

X={0,2,4}șiY={4,0,2}X = \{0, 2, 4\} \quad \text{și} \quad Y = \{4, 0, 2\}X={0,2,4}șiY={4,0,2}

Observăm că:

X=YX = YX=Y

Exemplul 4

Fie:

D={1,2,3}D = \{1, 2, 3\}D={1,2,3}

Toate submulțimile lui DDD sunt:

∅,{1},{2},{3},{1,2},{1,3},{2,3},D\varnothing, \{1\}, \{2\}, \{3\}, \{1, 2\}, \{1, 3\}, \{2, 3\}, D∅,{1},{2},{3},{1,2},{1,3},{2,3},D

Acestea formează mulțimea părților lui DDD:

P(D)={∅,{1},{2},{3},{1,2},{1,3},{2,3},D}P(D) = \{\varnothing, \{1\}, \{2\}, \{3\}, \{1, 2\}, \{1, 3\}, \{2, 3\}, D\}P(D)={∅,{1},{2},{3},{1,2},{1,3},{2,3},D}

Operații cu mulțimi​

Intersecția mulțimilor​

Definitie 1

Intersecția mulțimilor AAA și BBB este mulțimea formată din elementele comune mulțimilor AAA și BBB.

Scriem:

A∩B={x∣x∈A și x∈B}A \cap B = \{ x \mid x \in A \text{ și } x \in B \}A∩B={x∣x∈A și x∈B}

Două mulțimi a căror intersecție este mulțimea vidă se numesc mulțimi disjuncte.

Exemplu:

A={1,2,3,4,5},B={2,4,6}A = \{1, 2, 3, 4, 5\}, \quad B = \{2, 4, 6\}A={1,2,3,4,5},B={2,4,6}

Rezultă:

A∩B={2,4}A \cap B = \{2, 4\}A∩B={2,4}

Reuniunea mulțimilor​

Definite 2

Reuniunea a două mulțimi AAA și BBB este mulțimea formată din elementele care aparțin cel puțin uneia dintre mulțimile AAA sau BBB.

Scriem:

A∪B={x∣x∈A sau x∈B}A \cup B = \{ x \mid x \in A \text{ sau } x \in B \}A∪B={x∣x∈A sau x∈B}

Exemplu:

A={1,2,3,4,5},B={2,4,6}A = \{1, 2, 3, 4, 5\}, \quad B = \{2, 4, 6\}A={1,2,3,4,5},B={2,4,6}

Rezultă:

A∪B={1,2,3,4,5,6}A \cup B = \{1, 2, 3, 4, 5, 6\}A∪B={1,2,3,4,5,6}

Diferența a două mulțimi​

Definitie 3

Diferența dintre mulțimea AAA și mulțimea BBB este mulțimea formată din elementele lui AAA care nu aparțin lui BBB.

Scriem:

A−B={x∣x∈A și x∉B},B−A={x∣x∈B și x∉A}A - B = \{ x \mid x \in A \text{ și } x \notin B \}, \quad B - A = \{ x \mid x \in B \text{ și } x \notin A \}A−B={x∣x∈A și x∈/B},B−A={x∣x∈B și x∈/A}
Observație
A−B≠B−AA - B \ne B - AA−B=B−A
Exemplu
A={1,2,3,4,5},B={2,4,6}A = \{1, 2, 3, 4, 5\}, \quad B = \{2, 4, 6\}A={1,2,3,4,5},B={2,4,6}

Rezultă:

A−B={1,3,5},B−A={6}A - B = \{1, 3, 5\}, \quad B - A = \{6\}A−B={1,3,5},B−A={6}
