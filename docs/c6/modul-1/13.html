C6.2 - Mărimi invers proporționale

Definiție​

Definitie

Două mărimi se numesc invers proporționale dacă atunci când una crește (sau se micșorează) de un număr de ori, cealaltă se micșorează (respectiv crește) de același număr de ori.

Exemplu

Numărul de muncitori și numărul de zile în care finalizează o lucrare:

Nr. muncitoriNr. zile
86
163
412

Din tabel observăm că:

  • 816=168=416=63=36=124\dfrac{8}{16} = \dfrac{16}{8} = \dfrac{4}{16} = \dfrac{6}{3} = \dfrac{3}{6} = \dfrac{12}{4}168​=816​=164​=36​=63​=412​

  • 8⋅6=16⋅3=4⋅128 \cdot 6 = 16 \cdot 3 = 4 \cdot 128⋅6=16⋅3=4⋅12

Proprietăți​

Proprietatea 1

Raportul a două valori din prima mărime este egal cu inversul raportului valorilor corespunzătoare din cealaltă mărime.

a1a2=b2b1\frac{a_1}{a_2} = \frac{b_2}{b_1}a2​a1​​=b1​b2​​
Proprietatea 2

Produsul valorilor corespunzătoare din cele două mărimi este constant.

a1b1=a2b2=⋯=anbna_1 b_1 = a_2 b_2 = \cdots = a_n b_na1​b1​=a2​b2​=⋯=an​bn​

Mulțimi de numere​

Definitie

Fiind date două mulțimi
A={a1,a2,…,an},B={b1,b2,…,bn},A=\{a_1,a_2,\dots,a_n\},\quad B=\{b_1,b_2,\dots,b_n\},A={a1​,a2​,…,an​},B={b1​,b2​,…,bn​},
vom spune că între elementele acestora există o dependență invers proporțională (sunt invers proporționale) dacă

a1b1=a2b2=⋯=anbn,a_1 b_1 = a_2 b_2 = \cdots = a_n b_n,a1​b1​=a2​b2​=⋯=an​bn​,

sau, echivalent,

a1a2=b2b1=⋯=anbn.\frac{a_1}{a_2} = \frac{b_2}{b_1} = \cdots = \frac{a_n}{b_n}.a2​a1​​=b1​b2​​=⋯=bn​an​​.

Probleme rezolvate​

Problema 1

4 robinete umplu un bazin în 9 ore. În câte ore umplu bazinul 6 robinete?

Rezolvare

46=x9  ⟹  x=4⋅96=6 ore\frac{4}{6} = \frac{x}{9} \implies x = \frac{4 \cdot 9}{6} = 6\ \text{ore}64​=9x​⟹x=64⋅9​=6 ore
Problema 2

Determinați numerele a,b,ca, b, ca,b,c, știind că sunt invers proporționale cu 5, 3 și 15 și

a2+b2+c2abc=79\frac{a^2 + b^2 + c^2}{a b c} = \frac{7}{9}abca2+b2+c2​=97​

Rezolvare
{a,b,c}\{a,b,c\}{a,b,c} i.p. {5,3,15}\{5,3,15\}{5,3,15}

⇒5a=3b=15c  ⟹  a=3c, b=5c\Rightarrow 5a = 3b = 15c \implies a=3c,\ b=5c⇒5a=3b=15c⟹a=3c, b=5c

9c2+25c2+c23c⋅5c⋅c=35c215c3=73c=79\frac{9c^2 + 25c^2 + c^2}{3c \cdot 5c \cdot c} = \frac{35c^2}{15c^3} = \frac{7}{3c} = \frac{7}{9}3c⋅5c⋅c9c2+25c2+c2​=15c335c2​=3c7​=97​   ⟹  3c=9  ⟹  c=3, a=9, b=15\implies 3c = 9 \implies c = 3,\ a = 9,\ b = 15⟹3c=9⟹c=3, a=9, b=15
Problema 3

Determinați numerele a,b,c,d,ea, b, c, d, ea,b,c,d,e, știind că primele trei sunt direct proporționale cu 2, 3, 4, ultimele trei sunt invers proporționale cu 2, 3, 4 și a+b+c+d+e=246.a + b + c + d + e = 246.a+b+c+d+e=246.

Rezolvare

{a,b,c}\{a, b, c\}{a,b,c} d.p. {2,3,4}\{2,3,4\}{2,3,4}

  ⟹  a2=b3=c4\implies \dfrac{a}{2} = \dfrac{b}{3} = \dfrac{c}{4}⟹2a​=3b​=4c​ (1)

{c,d,e}\{c, d, e\}{c,d,e} i.p. {2,3,4}\{2,3,4\}{2,3,4}

  ⟹  2c=3d=4e\implies 2c = 3d = 4e⟹2c=3d=4e

  ⟹  c4=3d8=e2\implies \dfrac{c}{4} = \dfrac{3d}{8} = \dfrac{e}{2}⟹4c​=83d​=2e​

  ⟹  c4=d8/3=e2\implies \dfrac{c}{4} = \dfrac{d}{8/3} = \dfrac{e}{2}⟹4c​=8/3d​=2e​ (2)

Din (1) și (2):

a2=b3=c4=d83=e2=a+b+c+d+e2+3+4+83+2=24611+83=246⋅341=18.\frac{a}{2} = \frac{b}{3} = \frac{c}{4} = \frac{d}{\frac{8}{3}} = \frac{e}{2} = \frac{a + b + c + d + e}{2 + 3 + 4 + \frac{8}{3} + 2} = \frac{246}{11 + \frac{8}{3}} = \frac{246 \cdot 3}{41} = 18.2a​=3b​=4c​=38​d​=2e​=2+3+4+38​+2a+b+c+d+e​=11+38​246​=41246⋅3​=18.

Prin urmare:
