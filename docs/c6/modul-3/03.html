C18.1 - Mulțimea numerelor raționale

Numere Raționale

Definiție și notație​

NUMĂR RAȚIONAL: un număr xxx se numește număr rațional, dacă există o pereche de numere întregi (a,b)(a, b)(a,b) cu b≠0b \neq 0b=0, astfel încât x=abx = \dfrac{a}{b}x=ba​ (o împărțire neefectuată dintre două numere întregi).

Mulțimea acestor numere se notează cu Q\mathbb{Q}Q și este definită astfel:

Q={x∣∃a,b∈Z,b≠0, astfel ıˆncaˆt x=ab}\mathbb{Q} = \left\{ x \mid \exists a, b \in \mathbb{Z}, b \neq 0, \text{ astfel încât } x = \dfrac{a}{b} \right\}Q={x∣∃a,b∈Z,b=0, astfel ıˆncaˆt x=ba​}

Observații:​

  1. Are loc incluziunea N⊂Z⊂Q\mathbb{N} \subset \mathbb{Z} \subset \mathbb{Q}N⊂Z⊂Q (orice număr natural este și număr întreg, iar orice număr întreg este și rațional cu numitorul 1).
  2. Q∗=Q∖{0}\mathbb{Q}^* = \mathbb{Q} \setminus \{0\}Q∗=Q∖{0} este mulțimea numerelor raționale diferite de 0.
  3. Dacă a=0a=0a=0, atunci numărul rațional este egal cu 0.
  • Dacă aaa și bbb au același semn, atunci numărul rațional este pozitiv.
    • Exemplu: +3+7=+37,−3−7=+37\dfrac{+3}{+7} = +\dfrac{3}{7}, \quad \dfrac{-3}{-7} = +\dfrac{3}{7}+7+3​=+73​,−7−3​=+73​
  • Dacă aaa și bbb au semne diferite, atunci numărul rațional este negativ.
    • Exemplu: +3−7=−3+7=−37\dfrac{+3}{-7} = \dfrac{-3}{+7} = -\dfrac{3}{7}−7+3​=+7−3​=−73​

Forme de scriere a numerelor raționale​

Un număr rațional poate fi reprezentat prin fracții ordinare echivalente sau prin fracții zecimale finite sau infinite periodice.

În clasa a V-a am învățat că orice fracție ordinară se poate transforma în fracție zecimală și invers. Iată schema recapitulativă a acestor transformări:

1. Transformarea fracțiilor ordinare în fracții zecimale​

definitie

Se realizează prin împărțirea numărătorului la numitor (a:ba : ba:b).

a) Fracții zecimale finite​

Se obțin atunci când descompunerea numitorului conține doar factorii 2 sau 5.

Exemple
  • 114=2,75\dfrac{11}{4} = 2,75411​=2,75
  • 395=7,8\dfrac{39}{5} = 7,8539​=7,8
  • 325=0,1\dfrac{3}{25} = 0,1253​=0,1

b) Fracții zecimale infinite periodice simple​

Se obțin atunci când descompunerea numitorului conține factori diferiți de 2 și 5 (nu conține nici 2, nici 5).

Exemple
  • 23=0,666...=0,(6)\dfrac{2}{3} = 0,666... = 0,(6)32​=0,666...=0,(6)
  • 711=0,636363...=0,(63)\dfrac{7}{11} = 0,636363... = 0,(63)117​=0,636363...=0,(63)
  • 439=4,77777...=4,(7)\dfrac{43}{9} = 4,77777... = 4,(7)943​=4,77777...=4,(7)

c) Fracții zecimale infinite periodice mixte​

Se obțin atunci când descompunerea numitorului conține și alți factori pe lângă 2 sau 5.

Exemple
  • 176=2,83333...=2,8(3)\dfrac{17}{6} = 2,83333... = 2,8(3)617​=2,83333...=2,8(3)
  • 815=0,5333...=0,5(3)\dfrac{8}{15} = 0,5333... = 0,5(3)158​=0,5333...=0,5(3)
  • 5122=2,3181818...=2,3(18)\dfrac{51}{22} = 2,3181818... = 2,3(18)2251​=2,3181818...=2,3(18)

2. Transformarea fracțiilor zecimale în fracții ordinare​

a) Fracție zecimală finită​

La numărător scriem numărul dat fără virgulă, iar la numitor puterea lui 10 cu exponent egal cu numărul de zecimale.

Exemple

15,34=153410015,34 = \dfrac{1534}{100}15,34=1001534​ 0,123=12310000,123 = \dfrac{123}{1000}0,123=1000123​ 5,27=5271005,27 = \dfrac{527}{100}5,27=100527​ 21,9=2191021,9 = \dfrac{219}{10}21,9=10219​

b) Fracție zecimală infinită periodică simplă​

  • Dacă partea întreagă este 0, la numărător scriem perioada, la numitor atâtea cifre de 9 câte cifre are perioada.
  • Dacă partea întreagă este diferită de 0, scriem partea întreagă separat, obținând astfel un număr mixt.
Exemple

0,(13)=13990,(13) = \dfrac{13}{99}0,(13)=9913​ 2,(11)=211992,(11) = 2\dfrac{11}{99}2,(11)=29911​

Atenție! Partea întreagă nu o introducem la numărător: 21199≠211992\dfrac{11}{99} \neq \dfrac{211}{99}29911​=99211​.

c) Fracție zecimală infinită periodică mixtă​

  • Dacă partea întreagă este 0, la numărător scriem diferența dintre numărul format din toate cifrele părții zecimale și numărul format din cifrele perioadei (partea neperiodică), iar la numitor atâtea cifre de 9 câte cifre are perioada și atâtea de 0, câte cifre are partea neperiodică.
  • Dacă partea întreagă este diferită de 0, scriem partea întreagă separat.
Exemple

0,1(3)=13−190=12900,1(3) = \dfrac{13-1}{90} = \dfrac{12}{90}0,1(3)=9013−1​=9012​ 2,5(01)=2501−5900=24969002,5(01) = 2\dfrac{501-5}{900} = 2\dfrac{496}{900}2,5(01)=2900501−5​=2900496​

Observație: Nu există fracții zecimale cu perioadă 9.
