G1 - Unghiuri adiacente și bisectoarea unui unghi

Unghiuri adiacente​

Definiție

Două unghiuri proprii, care au același vârf, o latură comună situată în interiorul unghiului determinat de celelalte două, se numesc unghiuri adiacente.

Exemple​

  • ∡MON\measuredangle MON∡MON și ∡NOP\measuredangle NOP∡NOP din figura alăturată sunt adiacente.
  • ∡MON\measuredangle MON∡MON și ∡MOP\measuredangle MOP∡MOP nu sunt adiacente, deoarece latura comună nu este în interiorul unghiului determinat de celelalte două.

Bisectoarea unui unghi​

Definiție

Bisectoarea unui unghi propriu este semidreapta cu originea în vârful unghiului, inclusă în interiorul unghiului, și care formează cu laturile acestuia unghiuri congruente.

Construcția bisectoarei unui unghi​

a) Cu ajutorul raportorului​

Exemplu practic
  1. Măsurăm unghiul: m(∡MON)=70∘m(\measuredangle MON) = 70^\circm(∡MON)=70∘
  2. Calculăm jumătate din măsură: 70:2=3570 : 2 = 3570:2=35
  3. Așezăm raportorul, marcăm punctul SSS corespunzător măsurii de 35∘35^\circ35∘
  4. Cu ajutorul riglei, construim semidreapta cu originea în OOO care trece prin SSS
    → Aceasta este bisectoarea unghiului ∡MON\measuredangle MON∡MON.

b) Cu rigla negradată și compasul​

Pași de construcție
  1. Se desenează unghiul propriu ∡MON\measuredangle MON∡MON
  2. Cu acul compasului în OOO, se trasează un arc care taie laturile în punctele AAA și BBB
  3. Cu acul compasului în AAA, se trasează un arc în interiorul unghiului
  4. Cu acul compasului în BBB și aceeași deschidere, se taie ultimul arc în punctul PPP
  5. Unind punctele OOO și PPP, obținem bisectoarea unghiului ∡MON\measuredangle MON∡MON.

4. Aplicație​

Problemă

În figura de mai jos, m(∡AOC)=130∘m(\measuredangle AOC) = 130^\circm(∡AOC)=130∘, m(∡BOC)=50∘m(\measuredangle BOC) = 50^\circm(∡BOC)=50∘,
iar [OM)[OM)[OM) este bisectoarea unghiului ∡BOC\measuredangle BOC∡BOC.

Calculați:
m(∡AOB)m(\measuredangle AOB)m(∡AOB), m(∡BOM)m(\measuredangle BOM)m(∡BOM), m(∡AOM)m(\measuredangle AOM)m(∡AOM).

Rezolvare​

m(∡AOB)=m(∡AOC)−m(∡BOC)=130∘−50∘=80∘m(\measuredangle AOB) = m(\measuredangle AOC) - m(\measuredangle BOC) = 130^\circ - 50^\circ = 80^\circm(∡AOB)=m(∡AOC)−m(∡BOC)=130∘−50∘=80∘ m(∡BOM)=m(∡BOC):2=50∘:2=25∘m(\measuredangle BOM) = m(\measuredangle BOC) : 2 = 50^\circ : 2 = 25^\circm(∡BOM)=m(∡BOC):2=50∘:2=25∘ m(∡AOM)=m(∡AOB)+m(∡BOM)=80∘+25∘=105∘m(\measuredangle AOM) = m(\measuredangle AOB) + m(\measuredangle BOM) = 80^\circ + 25^\circ = 105^\circm(∡AOM)=m(∡AOB)+m(∡BOM)=80∘+25∘=105∘

R:
m(∡AOB)=80∘m(\measuredangle AOB) = 80^\circm(∡AOB)=80∘,
m(∡BOM)=25∘m(\measuredangle BOM) = 25^\circm(∡BOM)=25∘,
m(∡AOM)=105∘m(\measuredangle AOM) = 105^\circm(∡AOM)=105∘.
