# C11 - Compararea și ordonarea numerelor întregi

---

:::danger De Retinut
Fiecărui **număr întreg** îi corespunde pe **axa numerelor** un **punct**.  
Numărul asociat punctului se numește **abscisa (coordonata)** acestuia.
:::

:::note Exemplu
Pe axa numerelor, avem următoarele abscise:  
$A(-5)$, $B(-2)$, $O(0)$, $C(2)$, $D(5)$, $E(6)$.
:::

## Compararea numerelor întregi

:::tip Reguli importante
1. Dintre două numere întregi diferite, **mai mare este cel care se află în dreapta pe axă**.  
   Exemplu:  
   $-7 < -6 < -5 < -4$

2. Dintre două numere întregi **pozitive**, este mai mare cel care are **modulul mai mare**.  
   Exemplu:  
   $+9 > +2$, deoarece $9 > 2$

3. Dintre două numere întregi **negative**, este mai mare cel care are **modulul mai mic**.  
   Exemplu:  
   $-12 < -10$, deoarece $|{-12}| = 12 > 10 = |{-10}|$

4. **Orice număr pozitiv este mai mare decât orice număr negativ.**  
   Exemple:  
   $-5 < +1$, $+2 > -19$
:::

## Aplicații

### Exercițiul 1 — Compararea numerelor întregi

Comparați următoarele numere:

| Numere | Relație corectă |
|---------|----------------|
| $-76$ și $-9$ | $-76 < -9$ |
| $+98$ și $0$ | $+98 > 0$ |
| $0$ și $-87$ | $0 > -87$ |
| $-19$ și $-3$ | $-19 < -3$ |
| $+75$ și $-75$ | $+75 > -75$ |
| $+7$ și $7$ | $+7 = 7$ |
| $-76$ și $-67$ | $-76 < -67$ |
| $-456$ și $-564$ | $-456 > -564$ |
| $-1$ și $-121$ | $-1 > -121$ |
| $-93$ și $0$ | $-93 < 0$ |

---

### Exercițiul 2 — Ordonare crescătoare

Ordonăm crescător șirul de numere:

$$
+9,\ -98,\ -89,\ 89,\ 989,\ 99,\ -998,\ 0
$$

**Rezolvare:**

$$
-998,\ -98,\ -89,\ 0,\ +9,\ 89,\ 99,\ 989
$$

---

### Exercițiul 3 — Ordonare descrescătoare

Ordonăm descrescător șirul de numere:

$$
-43,\ -4300,\ -4003,\ +4,\ +430,\ -430,\ 403
$$

**Rezolvare:**

$$
+430,\ 403,\ +4,\ -43,\ -430,\ -4003,\ -4300
$$
