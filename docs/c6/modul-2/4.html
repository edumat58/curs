C12.2 - Înmulțirea și împărțirea numerelor întregi

Definiție

Produsul a două numere întregi nenule este numărul întreg care are:

  • modulul egal cu produsul modulelor factorilor;
  • semnul „+” dacă factorii au același semn;
  • semnul „–” dacă factorii au semne contrare.

Dacă cel puțin unul dintre factori este nul, atunci produsul este 0.

(−)⋅(+)(-)\cdot(+)(−)⋅(+)(−)(-)(−)

Proprietățile înmulțirii numerelor întregi​

Înmulțirea numerelor întregi are aceleași proprietăți ca și înmulțirea numerelor naturale:

  1. Asociativitate:
    (a⋅b)⋅c=a⋅(b⋅c),∀a,b,c∈Z(a\cdot b)\cdot c = a\cdot(b\cdot c), \quad \forall a,b,c\in\mathbb{Z}(a⋅b)⋅c=a⋅(b⋅c),∀a,b,c∈Z

  2. Comutativitate:
    a⋅b=b⋅a,∀a,b∈Za\cdot b = b\cdot a, \quad \forall a,b\in\mathbb{Z}a⋅b=b⋅a,∀a,b∈Z

  3. Element neutru:
    a⋅1=1⋅a=a,∀a∈Za\cdot 1 = 1\cdot a = a, \quad \forall a\in\mathbb{Z}a⋅1=1⋅a=a,∀a∈Z

  4. Distributivitate față de adunare și scădere:

    a⋅(b+c)=a⋅b+a⋅ca⋅(b−c)=a⋅b−a⋅ca\cdot(b+c) = a\cdot b + a\cdot c \\ a\cdot(b-c) = a\cdot b - a\cdot ca⋅(b+c)=a⋅b+a⋅ca⋅(b−c)=a⋅b−a⋅c

Multiplii unui număr întreg​

Definiție

Fie aaa și bbb două numere întregi.
Spunem că aaa este multiplul lui bbb (sau că aaa se divide cu bbb) dacă există un număr întreg ccc astfel încât:

a=b⋅ca = b \cdot ca=b⋅c

Scriem a:ba : ba:b sau b∣ab \mid ab∣a.
Numărul bbb se numește divizor al lui aaa.

Mulțimea multiplilor numărului întreg nnn se notează cu:

Mn={k⋅n∣k∈Z}M_n = \{k\cdot n \mid k \in \mathbb{Z}\}Mn​={k⋅n∣k∈Z}

Se observă că:

Mn=M−n,∀n∈ZM_n = M_{-n}, \quad \forall n \in \mathbb{Z}Mn​=M−n​,∀n∈Z

Exemple​

  1. M5={0,±5,±10,±15,±20,…,±5n,… }M_5 = \{0, \pm5, \pm10, \pm15, \pm20, \dots, \pm5n, \dots\}M5​={0,±5,±10,±15,±20,…,±5n,…}
  2. M−3={0,±3,±6,±9,±12,±15,…,±3n,… }M_{-3} = \{0, \pm3, \pm6, \pm9, \pm12, \pm15, \dots, \pm3n, \dots\}M−3​={0,±3,±6,±9,±12,±15,…,±3n,…}
  3. Dintre numerele 10,44,125,−108,253,120,−200,−302,62410, 44, 125, -108, 253, 120, -200, -302, 62410,44,125,−108,253,120,−200,−302,624, multiplii lui 444 sunt:

Împărțirea numerelor întregi​

Definiție

Fie a,b∈Za, b \in \mathbb{Z}a,b∈Z, cu b≠0b \neq 0b=0 și aaa un multiplu al lui bbb.
Câtul dintre aaa și bbb, notat a:ba : ba:b, este acel număr întreg ccc pentru care:

a=b⋅ca = b \cdot ca=b⋅c

aaa se numește deîmpărțit, iar bbb împărțitor.

Observații​

  • Regula semnelor este aceeași ca la înmulțire.
  • Valoarea absolută a câtului este egală cu câtul valorilor absolute.
  • Dacă deîmpărțitul este nul, atunci câtul este 0.
  • Dacă împărțitorul este nul, împărțirea nu are sens.

Divizorii unui număr întreg​

Definiție

Dacă aaa este un multiplu al lui bbb, atunci bbb este un divizor al numărului aaa.

Mulțimea divizorilor unui număr întreg nnn se notează cu:

Dn=D−n,∀n∈ZD_n = D_{-n}, \quad \forall n \in \mathbb{Z}Dn​=D−n​,∀n∈Z

Exemple​

  1. D6={±1,±2,±3,±6}D_6 = \{\pm1, \pm2, \pm3, \pm6\}D6​={±1,±2,±3,±6}
  2. D−15={±1,±3,±5,±15}D_{-15} = \{\pm1, \pm3, \pm5, \pm15\}D−15​={±1,±3,±5,±15}
  3. Divizorii numerelor:
    • 181818: divizori naturali ={1,2,3,6,9,18}=\{1, 2, 3, 6, 9, 18\}={1,2,3,6,9,18}, divizori întregi ={±1,±2,±3,±6,±9,±18}=\{\pm1, \pm2, \pm3, \pm6, \pm9, \pm18\}={±1,±2,±3,±6,±9,±18}
    • −37-37−37: divizori naturali ={1,37}=\{1, 37\}={1,37}, divizori întregi ={±1,±37}=\{\pm1, \pm37\}={±1,±37}
  4. Dintre numerele −9,+6,−12,14,21,−16,+24,−36,32-9, +6, -12, 14, 21, -16, +24, -36, 32−9,+6,−12,14,21,−16,+24,−36,32, divizorii lui 144144144 sunt:
