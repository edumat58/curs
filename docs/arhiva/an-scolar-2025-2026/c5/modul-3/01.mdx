# C19.1 - Fracții subunitare, echiunitare şi supraunitare 

---

## Fracții ordinare. Noțiuni introductive

## <span style={{backgroundColor: '#e65100', color: 'white', padding: '4px 8px', borderRadius: '4px'}}>Important</span>

:::tip
* Orice obiect care urmează a fi împărțit în mai multe părți de mărimi egale se va numi **întreg** sau **unitate** și se va reprezenta prin segmente sau figuri geometrice (dreptunghi, pătrat, cerc etc.)
:::

* O parte dintr-un întreg care a fost împărțit în părți de mărimi egale se numește **unitate fracționară**. Dacă luăm 0 părți egale dintr-un întreg, atunci nu luăm nimic, deci rezultatul va fi 0. În schimb, nu putem împărți niciodată un întreg în 0 părți egale. Cu alte cuvinte, numărătorul unei fracții poate fi 0, dar numitorul nu poate fi niciodată nul.

* Fracția $\dfrac{p}{100}$, unde $p$ este un număr, se poate scrie $p\%$ și se numește **procent**.

| Scriu | Citesc | Înțeleg |
| :---: | :---: | :---: |
| $p\%$ | $p$ la sută | $\dfrac{p}{100}$ |

## Clasificarea fracțiilor

:::danger De Reținut
| Denumire | Ce reprezintă? | Cum le recunosc? |
| :--- | :--- | :--- |
| Fracție subunitară | Este mai mică decât unitatea (un întreg) | numărător < numitor |
| Fracție echiunitară | Este egală cu unitatea (un întreg) | numărător = numitor |
| Fracție supraunitară | Este mai mare decât unitatea (un întreg) | numărător > numitor |
:::
