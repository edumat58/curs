# C19.3 - Compararea fracțiilor cu același numitor sau numărător

---

## Proprietăți ale fracțiilor

* **Orice fracție subunitară este cuprinsă între numerele naturale consecutive 0 și 1.**

  :::info[Exemplu]
  $0 < \dfrac{5}{9} < 1$
  :::

## <span style={{backgroundColor: '#e65100', color: 'white', padding: '4px 8px', borderRadius: '4px'}}>Exersează!</span>

:::note Exercițiul 1
Copiază și completează casetele cu unul din semnele $<, >, =$, pentru a obține propoziții adevărate.
:::

* **a)** $\dfrac{2}{7} \ \square \ \dfrac{3}{7}$; &emsp;
    **b)** $\dfrac{5}{11} \ \square \ \dfrac{4}{11}$; &emsp;
    **c)** $\dfrac{1}{4} \ \square \ \dfrac{1}{5}$; &emsp;
    **d)** $\dfrac{3}{5} \ \square \ \dfrac{3}{6}$; &emsp;
    **e)** $\dfrac{3}{3} \ \square \ \dfrac{7}{7}$; &emsp;
    **f)** $\dfrac{1}{6} \ \square \ \dfrac{1}{16}$; &emsp;
    **g)** $\dfrac{4}{44} \ \square \ \dfrac{4}{444}$.
:::

:::note Exercițiul 2
Scrie în ordine crescătoare fracțiile:
:::

* **a)** $\dfrac{3}{5}, \dfrac{2}{5}, \dfrac{4}{5}, \dfrac{7}{5}$; &emsp;
    
    **b)** $\dfrac{3}{2}, \dfrac{3}{7}, \dfrac{3}{5}, \dfrac{3}{10}$; &emsp;
    
    **c)** $\dfrac{1}{999}, \dfrac{1 \ 111}{999}, \dfrac{11 \ 111}{999}, \dfrac{11}{999}$; &emsp;
    
    **d)** $\dfrac{1 \ 000}{7}, \dfrac{1 \ 000}{11}, \dfrac{1 \ 000}{5}, \dfrac{1 \ 000}{13}$.

:::note Exercițiul 3
Determină numerele naturale nenule $n$ pentru care sunt adevărate inegalitățile:
:::

* **a)** $\dfrac{n}{6} < \dfrac{4}{6}$; &emsp;
    **b)** $\dfrac{2}{n} \ge \dfrac{2}{4}$; &emsp;
    **c)** $\dfrac{3}{14} < \dfrac{n}{14} < \dfrac{7}{14}$; &emsp;
    **d)** $\dfrac{11}{4} > \dfrac{11}{n} \ge \dfrac{11}{8}$.
