# C19.1 - Adunarea și scăderea numerelor raționale

---

* Pe mulțimea $\mathbb{Q}$, a numerelor raționale, se definește **suma a două numere raționale**.
* Dacă $a$ și $b$ sunt două numere raționale, **suma** acestor numere este un număr rațional, notat **$a+b$**, iar **$a$** și **$b$** se numesc **termenii sumei**.
* **Operația** prin care se obține suma a două numere raționale se numește **adunarea numerelor raționale**.

## <span style={{color: '#e65100', fontWeight: 'bold'}}>Proprietățile adunării numerelor raționale</span>

1.  **Comutativitatea:** $a+b = b+a$, oricare ar fi $a$ și $b$ numere raționale.
2.  **Asociativitatea:** $(a+b)+c = a+(b+c)$, oricare ar fi $a, b, c$ numere raționale.
3.  **0 este elementul neutru** la adunarea numerelor raționale: $a+0 = 0+a = a$, oricare ar fi numărul rațional $a$.
4.  **Orice număr rațional $a$ are un opus** notat $-a$, astfel încât: $a+(-a) = (-a)+a = 0$.

---

* Pe mulțimea $\mathbb{Q}$, a numerelor raționale, se definește **diferența a două numere raționale**.
  <span style={{color: 'red', fontWeight: 'bold'}}>$a - b = a + (-b)$</span> (se adună descăzutul cu opusul scăzătorului).
* **Operația** prin care se obține diferența a două numere raționale se numește **scăderea numerelor raționale**.

### <span style={{color: 'red', fontWeight: 'bold'}}>Reguli de eliminare a parantezelor</span>

1.  Dacă în fața parantezelor se află semnul **plus** sau niciun semn, atunci se elimină parantezele și semnul $+$ din față (dacă există), iar termenii din paranteze se scriu cu **semnele lor**.
2.  Dacă în fața parantezelor se află semnul **„ $-$ ”**, atunci se elimină parantezele și semnul minus din fața lor, iar termenii din paranteză se scriu cu **semn schimbat**.

---

Numerele raționale pot fi reprezentate prin fracții ordinare sau prin fracții zecimale. Astfel că, pentru a efectua calculele procedăm astfel:

## <span style={{color: '#e65100', fontWeight: 'bold'}}>1. Adunarea/Scăderea fracțiilor ordinare</span>

Dacă termenii unei sume sunt numere raționale reprezentate prin fracții ordinare atunci:
* se aduc fracțiile la același numitor (c.m.m.m.c. al numitorilor);
* se copiază numitorul comun și se adună numărătorii, aplicând regula semnelor de la adunarea numerelor întregi.

:::info[Exemple]
**1.a)** $\left(-\dfrac{11}{8}\right) + {}^{2)}\dfrac{3}{4} = -\dfrac{11}{8} + \dfrac{6}{8} = \dfrac{-11+6}{8} = -\dfrac{5}{8}$

**b)** $\left(-\dfrac{3}{2}\right) + \left(-\dfrac{2}{3}\right) = {}^{3)}\left(-\dfrac{3}{2}\right) + {}^{2)}\left(-\dfrac{2}{3}\right) = \left(-\dfrac{9}{6}\right) + \left(-\dfrac{4}{6}\right) = \dfrac{-9-4}{6} = -\dfrac{13}{6}$

**c)** ${}^{2)}\dfrac{5}{6} + \left(-{}^{4)}\dfrac{7}{3}\right) - \left(-{}^{3)}\dfrac{1}{4}\right) = \dfrac{10}{12} - \dfrac{28}{12} + \dfrac{3}{12} = \dfrac{10 - 28 + 3}{12} = \dfrac{13 - 28}{12} = -\dfrac{15^{(3}}{12} = -\dfrac{5}{4}$
:::

---

## <span style={{color: '#e65100', fontWeight: 'bold'}}>2. Adunarea/Scăderea fracțiilor zecimale finite</span>

Dacă termenii unei sume sunt numere raționale reprezentate prin fracții zecimale finite, operațiile se fac între modulele celor două numere raționale, iar semnul se stabilește după regulile cunoscute de la numere întregi.

:::info[Exemple]
**2.a)** $(-3,5) + (-2,4) = -(3,5 + 2,4) = -5,9$

**b)** $12,46 + (-3,2) = 12,46 - 3,2 = 9,26$

**c)** $(-10,53) + (+2,7) = -(10,53 - 2,7) = -7,83$

**d)** $-(-2,34) + (-5,2) = +2,34 - 5,2 = -(5,2 - 2,34) = -2,86$

**e)** $(-17,45) - (-3,8) = -17,45 + 3,8 = -13,65$
:::

---

## <span style={{color: '#e65100', fontWeight: 'bold'}}>3. Adunarea/Scăderea combinată (fracții ordinare și zecimale)</span>

Dacă termenii unei sume sunt numere raționale reprezentate prin fracții zecimale și prin fracții ordinare, mai întâi se transformă numerele raționale fie numai în fracții ordinare, fie numai în fracții zecimale și apoi se efectuează calculul ca în exemplele anterioare.

:::info[Exemple]
**3.a)** $\dfrac{2}{5} + (-0,5) = 0,4 + (-0,5) = -0,1$ (transformare în fracție zecimală)

*sau fracție ordinară și calcul cu fracții zecimale*

$\dfrac{2}{5} + (-0,5) = {}^{2)}\dfrac{2}{5} + \left(-\dfrac{5}{10}\right) = \dfrac{4}{10} + \left(-\dfrac{5}{10}\right) = -\dfrac{1}{10} = -0,1$ (transformare în fracție ordinară)

**b)** $(-0,7) + 3 - \dfrac{11}{3} + 1,(4) = -{}^{9)}\dfrac{7}{10} + {}^{90)}\dfrac{3}{1} - {}^{30)}\dfrac{11}{3} + {}^{10)}\dfrac{13}{9} = -\dfrac{63}{90} + \dfrac{270}{90} - \dfrac{330}{90} + \dfrac{130}{90} = \dfrac{7}{90}$

**c)** $1,2 - \dfrac{8}{3} + 0,(6) = {}^{3)}\dfrac{12}{10} - {}^{10)}\dfrac{8}{3} + {}^{3)}\dfrac{6}{9} = \dfrac{36}{30} - \dfrac{80}{30} + \dfrac{20}{30} = \dfrac{36 - 80 + 20}{30} = \dfrac{-24^{(6}}{30} = -\dfrac{4}{5}$

*Notă: Exemplul din imagine pare să aibă o eroare de calcul intermediar sau de afișare la numitori, dar rezultatul final $-\dfrac{4}{5}$ este coerent cu logica.*

**d)** $-10 - \left({}^{3)}\dfrac{1}{2} + {}^{2)}\dfrac{1}{3} + \dfrac{1}{6}\right) = -10 - \left(\dfrac{3}{6} + \dfrac{2}{6} + \dfrac{1}{6}\right) = -10 - \dfrac{6}{6} = -10 - 1 = -11$

*(calcul suma din paranteză)*

**sau**

$-{}^{6)}10 - {}^{3)}\dfrac{1}{2} - {}^{2)}\dfrac{1}{3} - \dfrac{1}{6} = -\dfrac{60}{6} - \dfrac{3}{6} - \dfrac{2}{6} - \dfrac{1}{6} = -\dfrac{66}{6} = -11$

*(eliminarea parantezei)*

**e)** $\left|{}^{3)}\dfrac{3}{4} - {}^{4)}\dfrac{4}{3}\right| - \dfrac{3}{20} = \left|\dfrac{9}{12} - \dfrac{16}{12}\right| - \dfrac{3}{20} = \left|\dfrac{-7}{12}\right| - \dfrac{3}{20} = {}^{5)}\dfrac{7}{12} - {}^{3)}\dfrac{3}{20} = \dfrac{35}{60} - \dfrac{9}{60} = \dfrac{26^{(2}}{60} = \dfrac{13}{30}$
:::
