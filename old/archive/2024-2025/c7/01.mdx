# 01 - Funcția de gradul I și graficul acesteia

---

## Gradul unei funcții

- gradul 0, **constantă**<Math display={true}>{String.raw`f(x) = a`}</Math> 
:::danger De Reținut
-  **gradul I**
<Math display={true}>{String.raw`\boldsymbol{f(x) = a \cdot x + b}`}</Math> 
:::
- gradul II
<Math display={true}>{String.raw`f(x) = a \cdot x^2 + b \cdot x + c`}</Math>  
- gradul III
<Math display={true}>{String.raw`f(x) = a \cdot x^3 + b \cdot x^2 + c \cdot x + d`}</Math> 
- gradul IV
<Math display={true}>{String.raw`f(x) = a \cdot x^4 + b \cdot x^3 + c \cdot x^2 + d \cdot x + e`}</Math>
<Math display={true}>{String.raw`\vdots`}</Math>
---
## Identificarea coeficienților și a necunoscutei
### Exemplu 1
:::note
Ecuație:
<Math display={true}>{String.raw`f(x) = x + 1 = 0`}</Math>
:::

Rezolvare:
<div style={{ display: 'flex', justifyContent: 'left' }}>
<Math display={true}>{String.raw`\begin{array}{l}
\begin{cases}
f(x) = ax + b \\[5pt]
f(x) = \;\ x + 1 = 0
\end{cases} \Rightarrow
\begin{cases}
\boldsymbol{a = 1} \\[5pt]
\boldsymbol{b = 1}
\end{cases}
\end{array}`}</Math>
</div>

<div style={{ display: 'flex', justifyContent: 'left' }}>
<Math display={true}>{String.raw`\begin{array}{l}
x + 1 = 0 \\[5pt]
x = 0 - 1 \\[5pt]
\boldsymbol{x = -1}
\end{array}`}</Math>  
</div>

**Coeficienți:**
- <Math>{String.raw`a = 1`}</Math>
- <Math>{String.raw`b = 1`}</Math>

**Necunoscuta:**
- <Math>{String.raw`x = -1`}</Math>

---

### Exemplu 2
:::note
Ecuație:
<Math display={true}>{String.raw`f(x) = -x + 3 = 0`}</Math>
:::
:::danger Atenție
- **Semnul lui `a`** este important.
Un număr negativ se obține prin **înmulțirea** numărului pozitiv cu **-1**.
- Opusul lui `a` este <Math display={true}>{String.raw`\boldsymbol{(-1) \cdot a = -a}`}</Math>
:::
Rezolvare:
<div style={{ display: 'flex', justifyContent: 'left' }}>
<Math display={true}>{String.raw`\begin{array}{l}
\begin{cases}
f(x) = \;\ a \quad \cdot x + b \\[5pt]
f(x) = (-1)\cdot x + 3 = 0
\end{cases} \Rightarrow
\begin{cases}
\boldsymbol{a = -1} \\[5pt]
\boldsymbol{b = 3}
\end{cases}
\end{array}`}</Math>
</div>

<div style={{ display: 'flex', justifyContent: 'left' }}>
<Math display={true}>{String.raw`\begin{array}{l}
-x + 3 = 0 \\[5pt]
-x = -3 \\[5pt]
\boldsymbol{x = 3}
\end{array}`}</Math>  
</div>

**Coeficienți:**
- <Math>{String.raw`a = -1`}</Math>
- <Math>{String.raw`b = 3`}</Math>

**Necunoscuta:**
- <Math>{String.raw`x = 3`}</Math>

---

### Exemplu 3
:::note
Ecuație:
<Math display={true}>{String.raw`f(x) = -2x + 6 = 0`}</Math>
:::

Rezolvare:
<div style={{ display: 'flex', justifyContent: 'left' }}>
<Math display={true}>{String.raw`\begin{array}{l}
\begin{cases}
f(x) = \;\ a \quad \cdot x + b \\[5pt]
f(x) = (-2)\cdot x + 6 = 0
\end{cases} \Rightarrow
\begin{cases}
\boldsymbol{a = -2} \\[5pt]
\boldsymbol{b = 6}
\end{cases}
\end{array}`}</Math>
</div>

<div style={{ display: 'flex', justifyContent: 'left' }}>
<Math display={true}>{String.raw`\begin{array}{l}
-2x + 6 = 0 \\[5pt]
-2x = -6 \\[5pt]
2x = 6 \\[5pt]
x = 6:2 \\[5pt]
\boldsymbol{x = 3}
\end{array}`}</Math>  
</div>

**Coeficienți:**
- <Math>{String.raw`a = -2`}</Math>
- <Math>{String.raw`b = 6`}</Math>

**Necunoscuta:**
- <Math>{String.raw`x = 3`}</Math>

---

### Exemplu 4
:::note
Ecuație:
<Math display={true}>{String.raw`f(x) = -5x + 2 = 5`}</Math>
:::

Rezolvare:
<div style={{ display: 'flex', justifyContent: 'left' }}>
<Math display={true}>{String.raw`\begin{array}{l}
\begin{cases}
f(x) = \;\ a \quad \cdot x + \quad b \\[5pt]
f(x) = (-5)\cdot x + (2 - 5) = 0
\end{cases} \Rightarrow
\begin{cases}
\boldsymbol{a = -5} \\[5pt]
\boldsymbol{b = -3}
\end{cases}
\end{array}`}</Math>
</div>

<div style={{ display: 'flex', justifyContent: 'left' }}>
<Math display={true}>{String.raw`\begin{array}{l}
-5x - 3 = 0 \\[5pt]
-5x = 3 \\[5pt]
5x = -3 \\[5pt]
x = (-3):5\\[5pt]
\boldsymbol{x = -\Large \frac{3}{5} \normalsize = -0,6}
\end{array}`}</Math>  
</div>

**Coeficienți:**
- <Math>{String.raw`a = -5`}</Math>
- <Math>{String.raw`b = -3`}</Math>

**Necunoscuta:**
- <Math>{String.raw`x = -\Large \frac{3}{5}`}</Math>

---

:::tip Observație
## Soluția ecuației
<Math display={true}>{String.raw`f(x) = 0 \Rightarrow ax + b = 0 \Rightarrow \boldsymbol{x = -\frac{b}{a}}`}</Math>
:::

## Graficul funcției

:::danger De Reținut
- **Funcția de gradul I** este o **dreaptă**.

### Forma generală  
<Math display={true}>{String.raw`\boldsymbol
{y = a \cdot x + b}\\[10pt]
\boldsymbol{y = f(x)}
`}</Math>
:::
### Pașii trasării graficului
:::note
Pentru funcția:  
<Math display={true}>{String.raw`f(x) = -x + 1`}</Math>
:::

**1. Se aleg 3 (minim 2) valori pentru <Math>{String.raw`x`}</Math>:**
  - <Math>{String.raw`x_1= - 1\rightarrow`}</Math> punctul de coordonate: <Math>{String.raw`\boldsymbol{(-1, y_1)}`}</Math>
  - <Math>{String.raw`x_2= 0\rightarrow`}</Math> punctul de coordonate: <Math>{String.raw`\boldsymbol{(0, y_2)}`}</Math>
  - <Math>{String.raw`x_3= 1\rightarrow`}</Math> punctul de coordonate: <Math>{String.raw`\boldsymbol{(1, y_3)}`}</Math>

**2. Se calculează coordonatele punctelor:**
    - <Math>{String.raw`y_1 = f(x_1) = -(-1) + 1 = 2 \Rightarrow \boldsymbol{\color{red}(-1, 2)}`}</Math>
    - <Math>{String.raw`y_2 = f(x_2) = (-0) + 1 = 1 \Rightarrow \boldsymbol{\color{green}(0, 1)}`}</Math>
    - <Math>{String.raw`y_3 = f(x_3) = (-1) + 1 = 0 \Rightarrow \boldsymbol{\color{orange}(1, 0)}`}</Math>
