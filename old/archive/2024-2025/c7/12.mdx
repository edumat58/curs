# 12 - Ecuația de gradul I cu o singură necunoscută  

---
:::tip Definiție
Este o ecuație care, după transformare, are forma $a \cdot x = b$, în care $a$ și $b$ sunt nr. cunoscute și $x$ este necunoscuta.
:::
---

## Transformări posibile

- dezvoltare sau factorizare a celor doi membri  
- adunarea aceleași sume algebrice în ambii membri  
- înmulțirea sau împărțirea celor doi membrii cu același număr ($\ne 0$)

:::info Important
Dacă $a \ne 0$, atunci ecuația $a \cdot x = b $ are ca soluție $x = \Large \frac{b}{a}$
:::

---
### Exemple

:::note 
Fie ecuația $4x+24=-6x+3$
:::

>adaugăm $(-24)$ în ambii membri:

$4x + 24 + (-24) = -6x + 3 + (-24)$

obținem $4x = -6x + 3 - 24$

>adaugăm $+6x$ în ambii membri:

$4x + 6x = -6x + 3 + (-24) + 6x$

$\Rightarrow 10x = -21$, unde `a = 10` și `b = -21`

x = $\Large \frac{-21}{10}$

$\boldsymbol{x = -2,1}$

---

:::note 
Fie ecuația $(2x+7)\cdot (3x-5) = 0$
:::

*Primul membru este un produs de 2 factori de gradul I; al doilea membru este nul.*

**Fiecare valoare care anulează unul dintre cei 2 factori din membrul întâi este o soluție a ecuației produs.**

>Primul factor:

$
  2x + 7 = 0 \Rightarrow 2x = -7 \Rightarrow \boldsymbol{x = -\Large \frac{7}{2}}
$

>Al doilea factor:

$
  3x - 5 = 0 \Rightarrow 3x = 5 \Rightarrow \boldsymbol{x = \Large \frac{5}{3}}
$

---

## Etapele de rezolvare
:::note
Într-un triunghi $ABC$ unghiul $\hat{A}$ este de $117^{\circ}$, iar unghiul $\hat{C}$ este dublul unghiului $\hat{B}$. Care este măsura unghiurilor $\hat{B}$ și $\hat{C}$?
:::

**1. Alegem necunoscuta**
>Folosim o literă pentru desemnarea necunoscutei

Notăm cu `x` măsura unghiului $\hat{B}$

---

**2. Precizăm unitățile și condițiile impuse de necunoscută**

Unitate de măsură: $^{\circ}$ (grade)

0 < x < 180

---

**3. Punerea în ecuație a problemei**
$$ 
\hat{C} = 2 \cdot \hat{B} \Rightarrow 
\hat{C} = 2 \cdot x
$$
> Spunem că suma unghiurilor este egală cu $180^{\circ}$

$
\hat{A} + \hat{B} + \hat{C} = 180^{\circ}, unde \\[8pt]
\begin{cases}
\hat{A} = 117^{\circ}\\
\hat{B} = x\\
\hat{C} = 2x\\
\end{cases}
\\[8pt]
\boldsymbol{\hat{A} + \hat{B} + \hat{C} = 117^{\circ} + x + 2x}
$

---

**4. Rezolvarea ecuației**

$
117^{\circ} + x + 2x = 180^{\circ} \Rightarrow\\[8pt]
117^{\circ} + 3x = 180^{\circ}\\[8pt]
3x = 180^{\circ} - 117^{\circ}\\[8pt]
3x = 63^{\circ}\\[8pt]
\boldsymbol{x = 63 : 3 = 21^{\circ}}
$
